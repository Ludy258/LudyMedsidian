# 血浆脂蛋白（Plasma Lipoproteins）

## 📌 定义

血浆脂蛋白是**脂质（TG、胆固醇、磷脂）** 与**载脂蛋白（apolipoprotein）** 在血浆中的可溶性复合物，负责脂质的**运输和分配**。按密度从小到大分为四大类：**CM → VLDL → LDL → HDL**。

> 🔑 脂蛋白的核心矛盾：脂质不溶于水，必须包裹在载脂蛋白"外壳"中才能在血液中运输。

---

## 🔬 一、脂蛋白的结构

所有脂蛋白结构模式相同：

```
                 ┌────────────────┐
                 │   核心         │
                 │ TG + 胆固醇酯  │ ← 疏水性脂质
                 └────────┬───────┘
       ┌───────────┬──────┴──────────────┐
       │           │                     │
   载脂蛋白     磷脂单层             游离胆固醇
   (Apo)       (亲水头向外)        (调节膜流动性)
```

---

## 🔬 二、四大脂蛋白分类对比

| 性质         | CM（乳糜微粒）            | VLDL（极低密度）           | LDL（低密度）         | HDL（高密度）        |
| :--------- | :------------------ | :------------------- | :--------------- | :-------------- |
| **密度**     | <0.95               | 0.95~1.006           | 1.019~1.063      | 1.063~1.210     |
| **电泳位置**   | 原点（不移动）             | 前β                   | **β**            | α               |
| **核心成分**   | **TG（85~95%）**      | **TG（55~65%）**       | **胆固醇酯（40~50%）** | 胆固醇酯+磷脂         |
| **主要载脂蛋白** | ApoB-48, ApoC, ApoE | ApoB-100, ApoC, ApoE | **ApoB-100**     | ApoA-I, ApoA-II |
| **合成场所**   | **小肠**（肠黏膜细胞）       | **肝脏**               | 由VLDL在血浆中转化      | 肝脏+小肠           |
| **功能**     | 运输外源性TG（膳食脂肪）       | 运输内源性TG（肝合成）         | **将胆固醇运送至外周组织**  | **将胆固醇从外周运回肝脏** |
| **与AS的关系** | **CM残余颗粒可致AS**     | **↑→中度致AS**（含IDL）    | **↑→AS高风险** ⭐⭐⭐  | **↑→保护** ⭐⭐⭐    |
| **临床参考**   | —                   | —                    | 理想<2.6 mmol/L    | 理想>1.0 mmol/L   |

## 🔬 三、载脂蛋白（Apolipoprotein）

| 载脂蛋白 | 来源 | 功能 |
|:---------|:-----|:-----|
| **ApoA-I** | 肝/肠 | HDL主要蛋白；激活**LCAT**（促进胆固醇酯化） |
| **ApoB-48** | 小肠 | CM的结构蛋白 |
| **ApoB-100** | 肝脏 | VLDL和LDL的结构蛋白；**介导LDL与LDL受体结合** |
| **ApoC-II** | 肝 | 激活**LPL**（脂蛋白脂肪酶）→ 水解TG |
| **ApoE** | 肝 | 介导CM残余和VLDL残余被肝清除；与Alzheimer病相关 |

## 🔬 四、代谢途径

### 外源性途径（膳食脂肪）

```
肠摄入膳食脂肪 → 在肠黏膜细胞合成CM
    ↓ 经淋巴→血液
CM 进入毛细血管 → ApoC-II激活LPL → 水解CM中的TG
    ↓
CM残余颗粒 → 被肝脏ApoE受体识别 → 肝内降解
```

### 内源性途径（肝合成脂肪）

```
肝脏合成并分泌VLDL
    ↓
VLDL进入毛细血管 → LPL水解TG → VLDL缩小
    ↓
VLDL → IDL（中间密度脂蛋白）
    ↓
IDL继续代谢 → 约50%被肝摄取；约50%转化为**LDL**
    ↓
LDL → 通过ApoB-100与肝及外周组织的LDL受体结合 → 内含物释放
```

### 逆向胆固醇转运（HDL—保护机制）

```
外周组织细胞（包括血管壁巨噬细胞和泡沫细胞）
    ↓
胆固醇流出 → ApoA-I结合胆固醇 → **HDL**
    ↓ ← LCAT（被ApoA-I激活）将胆固醇酯化
HDL成熟（含大量胆固醇酯）
    ↓
途径一：HDL将胆固醇酯转运至VLDL/LDL（通过CETP）→ 最终回肝
途径二：HDL直接与肝SR-BI受体结合 → 胆固醇释放至肝脏
    ↓
肝脏将胆固醇排出（胆汁酸+游离胆固醇）
```

> 🔑 **逆向胆固醇转运是HDL抗AS的核心机制**——HDL从泡沫细胞"吸走"胆固醇并送回肝脏代谢。HDL水平越高，血管壁胆固醇清除能力越强。

## 🔬 五、LDL受体与高胆固醇血症

### LDL受体途径

```
LDL → ApoB-100 → 与肝细胞表面**LDL受体**结合
    ↓ 受体介导内吞
内吞体 → 溶酶体 → LDL降解 + 胆固醇释放
    ↓
细胞内胆固醇↑ → ①下调LDL受体合成 ②抑制HMG-CoA还原酶 ③激活ACAT
```

> 🔑 这是**负反馈调节**——细胞内胆固醇足够时自动减少LDL摄取。当LDL受体缺陷或数量不足时→LDL在血浆中滞留→易于被氧化→进入动脉内膜→AS。

### 家族性高胆固醇血症

| 类型 | 基因缺陷 | LDL受体活性 | 血浆LDL | AS发生 |
|:-----|:---------|:----------:|:-------:|:------:|
| **杂合子(1:500)** | LDLR突变 | ~50% | ↑↑ | 30~40岁发生冠心病 |
| **纯合子(1:1,000,000)** | LDLR突变 | <10% | ↑↑↑↑ | 儿童期即可发生心梗 |

## 💊 临床药物靶点

| 药物类别 | 靶点 | 机制 | 效应 |
|:---------|:-----|:-----|:-----|
| **他汀类** | **HMG-CoA还原酶** | 抑制胆固醇合成 → 肝内胆固醇↓ → LDL受体合成↑ → 血浆LDL清除↑ | LDL↓↓ |
| **依折麦布** | NPC1L1（肠） | 抑制肠胆固醇吸收 | LDL↓ |
| **PCSK9抑制剂** | PCSK9 | PCSK9可降解LDL受体；抑制PCSK9 → LDL受体↑ → LDL清除↑ | LDL↓↓↓ |
| **贝特类** | PPARα | 促进VLDL清退 + HDL↑ | TG↓↓, HDL↑ |
| **烟酸** | HCAR2 | 抑制脂肪组织脂解 → VLDL合成↓ | HDL↑ |

## ❗ 易混点

- 🚨 **LDL ≠ β-脂蛋白**，HDL ≠ α-脂蛋白——前者是按密度分，后者是电泳分，两者对应但概念不同
- 🚨 **TG不直接致AS**——但高TG常伴随VLDL↑ → VLDL可转化为LDL，兼有致AS性
- 🚨 **ApoB-48（小肠）≠ ApoB-100（肝脏）**——同一个基因的两种编辑形式，记住48（48%全长）在CM，100（全长）在VLDL/LDL
- 🚨 **他汀升LDL受体 → LDL降**，他不是降LDL本身，而是增加LDL的清除能力
- 🚨 **PCSK9抑制剂是最新的强效降LDL药**——PCSK9天然作用是促进LDL受体降解，阻断PCSK9 = 保护LDL受体
