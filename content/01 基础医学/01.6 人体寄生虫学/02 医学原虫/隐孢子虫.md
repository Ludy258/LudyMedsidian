# 隐孢子虫（*Cryptosporidium* spp.）— 隐孢子虫病

## 📌 定义
- **微小隐孢子虫**（*Cryptosporidium parvum*）和**人隐孢子虫**（*C. hominis*）是主要致病虫种
- **细胞内寄生，胞外定位**（寄生于小肠上皮细胞微绒毛区域）
- 引起**隐孢子虫病**（cryptosporidiosis）
- **机会性致病原虫**：免疫正常→自限性腹泻；AIDS→**霍乱样腹泻（可致死）**
- **水源性暴发**的最常见病原体之一

---

## 🔬 形态

| 阶段 | 大小 | 特征 |
|:----|:----|:------|
| **卵囊（oocyst）** | **4~6μm** | 圆形/椭圆形；**改良抗酸染色→红色**；含4个裸露子孢子 |
| **子孢子（sporozoite）** | 1~2μm | 新月形，位于卵囊内 |
| **滋养体/裂殖体/配子体** | — | 寄生于宿主细胞微绒毛内（纳虫空泡） |

---

## 🔄 生活史

### 单宿主生活史（完全在同一宿主体内完成）

```
食入孢子化卵囊（水源/食物）
    ↓ 胃液+胰酶作用
子孢子脱囊 → 侵入小肠上皮细胞微绒毛（纳虫空泡内）
    ↓ 无性生殖（裂体增殖）
I型裂殖体 → 裂殖子 → 再感染邻近细胞（自身感染循环）
    ↓
II型裂殖体 → 裂殖子 → 配子生殖（雌雄配子）
    ↓
合子 → 卵囊（2种类型）
    ├── 厚壁卵囊（80%）→ 随粪便排出 → 外界感染源
    └── 薄壁卵囊（20%）→ 在肠内脱囊 → **自身感染**（自体感染循环）
```

### 关键要点
- **感染阶段**：孢子化卵囊
- **致病阶段**：裂体增殖（破坏肠上皮微绒毛）
- **自身感染机制**：薄壁卵囊 → 免疫低下者**迁延难愈**
- **无蚊虫/动物宿主必要**：人与人直接粪-口传播

### 传播途径
| 途径 | 说明 |
|:----|:------|
| **水源传播 🥇** | 饮用水/游泳池/娱乐用水污染 | 最常见暴发方式 |
| 粪-口传播 | 人传人（家庭/托幼机构）、动物传人 | — |
| 食源性 | 受污染食物 | — |
| 旅行相关 | 发展中国家感染风险高 | — |

---

## ⚙️ 致病机制

### 肠道病理链

```
子孢子侵入肠上皮微绒毛 → 纳虫空泡内增殖
    ↓
微绒毛萎缩、融合、变短 → 肠黏膜表面积↓
    ↓
刷状缘酶（乳糖酶/蔗糖酶等）活性↓ → 双糖吸收障碍
    ↓
Na⁺-依赖性葡萄糖吸收↓ + Cl⁻分泌↑ → 渗透性+分泌性腹泻

免疫正常：IFN-γ+CD4⁺T细胞 → 清除（1~2周自限）
免疫低下（CD4<200）：无法清除 → 大量薄壁卵囊自身感染 → 迁延不愈
```

### 病理生理特点
- **小肠为主**（空肠最重），重症可累及胃、大肠、胆道、呼吸道
- **不侵入黏膜下**（细胞内但胞外），故无溃疡/出血

---

## 🩺 临床表现

| 人群 | 表现 | 特点 |
|:----|:----|:------|
| **免疫正常者** | **自限性水样腹泻**（1~2周） | 量多、无血无脓，伴腹痛恶心低热 |
| **婴幼儿** | 腹泻+脱水 | 发展中国家儿童腹泻重要病因 |
| **AIDS患者 CD4<200 🚨** | **霍乱样大量水样便（3~10L/天）** | 迁延不愈→恶病质→致死 |
| **AIDS患者 CD4<50** | 播散：胆道（硬化性胆管炎）、肺（咳嗽气促） | 预后极差 |

### AIDS患者隐孢子虫病演进

```
CD4正常 → 自限性腹泻（同免疫正常）
CD4 200~350 → 腹泻迁延（数周~数月）
CD4 100~200 → 慢性腹泻（体重下降+吸收不良）
CD4 <100 → 霍乱样腹泻 + 胆道/呼吸道受累 → 死亡率高
```

## 🔬 检查

| 方法 | 适用 | 说明 |
|:----|:----|:------|
| **改良抗酸染色 🥇** | 粪便 | 卵囊染为**红色/粉红色**，背景蓝色（改良Ziehl-Neelsen法） |
| **金胺-酚染色** | 荧光镜检 | 卵囊呈**亮黄色荧光**（敏感度可高于抗酸染色） |
| **金胺-酚+改良抗酸复染** | 提高特异性 | 双重染色法 |
| **ELISA/IFA** | 粪便检测抗原 | 灵敏度高，适合批量筛查 |
| **PCR** | 粪便 | 敏感性/特异性最高；可鉴别虫种 |
| **肠道活检** | 内镜 | 小肠活检见虫体附着微绒毛 |

> 💡 **粪便查卵囊要点**：
> - 改良抗酸染色是常规方法
> - 卵囊4~6μm（比一般细菌大，比一般原虫小）→ **需仔细鉴别**
> - 免疫正常者排卵囊量少→可能需要浓集法提高检出率

## 💊 治疗

| 患者类型 | 方案 | 说明 |
|:---------|:----|:------|
| **免疫正常者** | **支持治疗**（补液） | 自限性，通常1~2周自愈 |
| **免疫正常（症状重）** | **硝唑尼特（Nitazoxanide）🥇** | **唯一FDA批准**，500mg bid×3天 |
| **AIDS/免疫低下者** | **HAART（ART）🥇** | **恢复CD4→免疫功能重建→清除虫体** |
| AIDS辅助 | 硝唑尼特+对症止泻 | 疗效有限（免疫未恢复时） |
| 胆道感染 | ERCP+括约肌切开 | 缓解梗阻 |

> 🚨 **免疫低下者的治疗关键在恢复免疫**：硝唑尼特在CD4低的AIDS患者中效果有限
> 🚨 **HAART时代**隐孢子虫病发病率明显下降（CD4恢复是关键）

### 对症支持
- 口服补液盐/静脉补液（纠正脱水+电解质紊乱）
- 营养支持（要素饮食）
- 止泻药（洛哌丁胺）— 慎用于重症

---

## 🛡️ 预防

| 措施 | 说明 |
|:----|:------|
| **安全饮水 🥇** | 煮沸（最可靠）、过滤（孔径<1μm可滤除卵囊） |
| 注意个人卫生 | 便后洗手（托幼机构重点） |
| 避免接触腹泻动物 | 尤其是犊牛（C. parvum动物源） |
| **AIDS患者** | 避免饮用自来水、生水；避免接触宠物粪便 |
| **泳池卫生** | 腹泻者勿入泳池（卵囊耐氯❗） |

> ⚠️ **隐孢子虫卵囊的抵抗力**：
> - 对氯消毒**高度耐受**（氯化消毒无法杀灭）
> - 煮沸可灭活
> - 可在外界存活数月

---

## 🌍 流行

| 特征 | 说明 |
|:----|:------|
| **分布** | 全球性，发展中国家感染率更高 |
| **主要虫种** | C. parvum（人兽共患）+ C. hominis（人际传播） |
| **季节性** | 夏秋季多见 |
| **高危人群** | 婴幼儿、AIDS患者、旅行者、托幼机构儿童 |
| **著名暴发** | 1993年Milwaukee（美国·40万感染）— 最大水源性暴发 |

---

> 💡 **临床推理链**：水样腹泻（>1周）+ 免疫正常→粪便改良抗酸染色查见红色卵囊（4~6μm）→ 确诊隐孢子虫病 → 自限性 → 补液/硝唑尼特（症状重）。**AIDS患者**：CD4<100 + 霍乱样腹泻 + 消瘦→粪便查卵囊 → 确诊 → **启动HAART是关键** + 硝唑尼特辅助
