# 非洲锥虫（*Trypanosoma brucei*）— 睡眠病

## 📌 定义
- 布氏冈比亚锥虫（*T. b. gambiense*）和布氏罗得西亚锥虫（*T. b. rhodesiense*）引起**非洲锥虫病**（**睡眠病**）
- 通过**舌蝇（采采蝇）** 叮咬传播

## 🔬 两种亚种对比

| 特征 | 冈比亚锥虫 | 罗得西亚锥虫 |
|:----|:----------|:----------|
| **流行地区** | 西非/中非（河流湖泊区） | 东非（草原灌木区） |
| **传染源** | **人**（病人） | **动物**（羚羊等野生动物/牛） |
| **病程** | **慢性**（数月至数年） | **急性**（数周至数月） |
| **CNS受累** | 较晚 | 较早 |
| **预后** | 相对较好 | 较差，死亡率高 |

## 🔄 生活史

```
舌蝇叮咬感染者 → 吸入锥鞭毛体
    ↓ 在蝇中肠
锥鞭毛体 → 前循环型 → 二分裂繁殖 → 移行至唾液腺
    ↓
转变为上鞭毛体 → 短粗型感染性锥鞭毛体
    ↓ 舌蝇再次叮咬人
锥鞭毛体随唾液注入人体 → 局部繁殖 → 入血 → 播散全身 → 晚期入CNS
```

- **传播媒介**：**舌蝇（采采蝇，*Glossina*）**
- **感染阶段**：后循环型锥鞭毛体（唾液腺）

## ⚙️ 致病机制

### 抗原变异（核心机制）
- 锥虫表面糖蛋白（VSG）**周期性变异**
- 逃避宿主免疫 → 宿主产生新抗体 → 虫体再次变异
- 虫血症呈**波浪式起伏**

### 免疫病理
- 免疫复合物沉积 → 血管炎、肾炎
- 细胞因子风暴 → 发热、组织损伤
- CNS浸润 → 脑膜脑炎

## 🩺 临床表现

| 分期 | 表现 |
|:----|:------|
| **初发反应期** | 叮咬处**锥虫下疳**（质硬无痛性结节），2~3周消退 |
| **血淋巴期（早期）** | 间歇性发热、全身浅表淋巴结肿大（**颈后三角→Winterbottom征**）、皮疹、心肌炎 |
| **脑膜脑炎期（晚期）** | 头痛、嗜睡、性格改变、**昼夜节律颠倒**（夜间失眠+白天嗜睡）、震颤、共济失调→昏迷→死亡 |

## 🔬 检查

| 方法 | 说明 |
|:----|:------|
| **血涂片** | 薄/厚血膜查锥鞭毛体 |
| **淋巴结穿刺** | 查锥鞭毛体 |
| **脑脊液检查** | 晚期查锥鞭毛体 |
| **CATT（卡片凝集试验）** | 现场筛查 |
| **PCR** | 检测锥虫DNA |

## 🆚 鉴别诊断
| 疾病 | 鉴别要点 |
|:----|:---------|
| **疟疾** | 周期性发热，血涂片查见疟原虫 |
| **结核病** | 结核中毒症状，TST(+) |
| **淋巴瘤** | 淋巴结活检病理特征 |
| **病毒性脑炎** | 脑脊液病毒学(+) |
| **HIV脑病** | HIV抗体(+)，CD4↓ |

## 💊 治疗

| 分期 | 药物 | 说明 |
|:----|:----|:------|
| **早期（血淋巴期）** | **苏拉明（Suramin）🥇** | 首选，对CNS渗透差 |
| | 喷他脒 | 替代 |
| **晚期（脑膜脑炎期）** | **美拉胂醇（Melarsoprol）** | 含砷可透过血脑屏障，毒性大 |
| | 依氟鸟氨酸 | 对冈比亚锥虫有效，毒性较小 |
| | **NECT**（硝呋莫司+依氟鸟氨酸） | **WHO推荐方案** |

## 🛡️ 预防
- 消灭舌蝇（杀虫剂、诱捕器）
- 避免进入流行区灌木丛
- 穿防护服、使用驱虫剂
