# 美洲锥虫（*Trypanosoma cruzi*）— 恰加斯病

## 📌 定义
- 枯氏锥虫（*Trypanosoma cruzi*）引起**美洲锥虫病**（**恰加斯病**）
- 属**粪源性锥虫**，由**锥蝽**（*Triatoma*）传播

## 🔬 三种形态

| 形态 | 特点 |
|:----|:------|
| **无鞭毛体** | 球形/卵圆形，寄生于**组织细胞内** |
| **上鞭毛体** | 纺锤形，寄生于**锥蝽后肠** |
| **锥鞭毛体** | 弯曲，游离于**血液**或锥蝽后肠 |

## 🔄 生活史

```
锥蝽叮咬感染者 → 吸入含锥鞭毛体血液
    ↓ 在锥蝽后肠
锥鞭毛体 → 上鞭毛体（二分裂） → 后循环型锥鞭毛体
    ↓ 锥蝽再次叮咬时排出含虫粪便
虫体经叮咬伤口或黏膜侵入人体
    ↓
局部组织 → 无鞭毛体 → 侵入细胞（巨噬细胞/肌细胞）→ 繁殖 → 破裂 → 再感染
```

- **传播媒介**：**锥蝽**（粪源性传播）
- **主要途径**：锥蝽叮咬后**排粪**接触伤口/黏膜
- **其他途径**：输血、母婴垂直、食入污染食物
- **流行地区**：中南美洲

> 🚨 **注意**：非洲锥虫是**唾液性传播**（舌蝇叮咬注入），美洲锥虫是**粪源性传播**（锥蝽排粪污染伤口）— 传播机制不同！

## ⚙️ 致病机制

| 分期 | 机制 |
|:----|:------|
| **急性期** | 虫体局部繁殖→炎症→血行播散 |
| **慢性期** | 免疫介导心肌损伤 + 神经节细胞破坏→心肌纤维化/巨食管/巨结肠 |

## 🩺 临床表现

### 急性期
| 体征 | 特点 |
|:----|:------|
| **Chagoma** | 叮咬处无痛性结节（玫瑰疹样） |
| **Romana征** 🥇 | **单侧眼睑水肿 + 结膜炎**（虫体经结膜感染）|
| 全身症状 | 发热、头痛、肌痛、肝脾肿大 |

### 慢性期
| 类型 | 表现 |
|:----|:------|
| **心脏型** | 心律失常、心力衰竭、血栓栓塞（**最常见死因**） |
| **消化道型** | **巨食管症**（吞咽困难）、**巨结肠症**（便秘） |
| **混合型** | 心脏+消化道症状 |

## 🔬 检查

| 方法 | 适用 |
|:----|:------|
| **血涂片** | 急性期查锥鞭毛体 |
| **浓集法** | 提高检出率 |
| **动物接种** | 敏感方法 |
| **ELISA/IFA/IHA** | **慢性期主要诊断方法**（检测抗体） |
| **PCR** | 检测锥虫DNA |

## 🆚 鉴别诊断
| 疾病 | 鉴别要点 |
|:----|:---------|
| **扩张型心肌病** | 无锥虫感染证据，免疫学(-) |
| **冠心病** | 危险因素、冠脉造影特征 |
| **特发性巨食管症** | 无锥虫感染证据 |
| **先天性心脏病** | 心脏超声特征 |

## 💊 治疗

| 药物 | 说明 |
|:----|:------|
| **苄硝唑（Benznidazole）🥇** | **首选**，急性期及慢性早期 |
| 硝呋莫司（Nifurtimox） | 替代用药 |
| ⚠️ 慢性晚期（器质性病变） | 抗虫治疗效果差 |

**对症治疗**：
- 心律失常→起搏器
- 心力衰竭→强心利尿扩血管
- 巨食管症→扩张术/手术
- 巨结肠症→手术

## 🛡️ 预防
- 改善住房条件，消灭锥蝽栖息场所
- 杀虫剂喷洒
- 输血筛查

> 💡 **临床推理链**：中南美洲居住/旅行史 + 锥蝽叮咬史 + Romana征 → 疑诊恰加斯病 → 血涂片/免疫学(+) → 确诊 → 苄硝唑治疗 → 定期心脏评估
