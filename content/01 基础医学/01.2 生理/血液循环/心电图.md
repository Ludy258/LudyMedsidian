# 心电图（Electrocardiogram, ECG/EKG）

## 📌 定义

心电图是将**体表电极**记录的整个心脏**综合动作电位**随时间变化的曲线。反映的是**电活动**（兴奋的产生与传导），不直接反映收缩力。

---

## 🔬 一、正常心电图各波与间期

```
        R
       /\
      /  \        T
     /    \      /\
    /      \    /  \
───/────────\──/────\───
    P       \/      U
            S
            Q
|←PR→|←QRS→|←ST→|←QT→|
```

| 波/段 | 生理意义 | 正常时长/幅度 | 对应心脏活动 |
|:------|:--------|:------------|:------------|
| **P波** | 心房去极化 | ≤0.11s, <0.25mV | 心房兴奋 |
| **PR段** | 房室结传导 | 等电位线 | 冲动过房室结 |
| **PR间期** | = P波 + PR段 | 0.12~0.20s | 心房去极→心室去极开始 |
| **QRS波群** | 心室去极化 | 0.06~0.10s | 心室兴奋 |
| **ST段** | 心室平台期 | 等电位线 | 心室全去极→复极开始前 |
| **T波** | 心室复极化 | 与QRS同方向 | 心室复极 |
| **QT间期** | = QRS + ST + T | HR校正后0.36~0.44s | 心室去极+复极全程 |
| **U波** | 浦肯野纤维复极(?) | 低振幅、T波后 | 可见于低钾 |

## 🔬 二、导联系统

### 标准12导联

| 类别 | 导联 | 电极位置 | 视角 |
|:----:|:-----|:---------|:-----|
| **双极肢体** | I, II, III | 两肢体间电位差 | 额面 |
| **加压肢体** | aVR, aVL, aVF | 单肢体 vs 零电位 | 额面 |
| **胸导联** | V₁~V₆ | 胸壁6个特定位置 | 水平面 |

### 心脏解剖与导联对应

| 心脏区域 | 对应导联 | 供血动脉 |
|:---------|:---------|:---------|
| **下壁** | II, III, aVF | 右冠脉(RCA) |
| **前间壁** | V₁~V₃ | 左前降支(LAD) |
| **前壁** | V₂~V₄ | LAD |
| **侧壁** | I, aVL, V₅~V₆ | 回旋支(LCx) |
| **高侧壁** | I, aVL | LCx |

## 🔬 三、心电轴

| 电轴 | QRS方向(I+III) | 常见原因 |
|:----:|:-------------:|:---------|
| 正常 | I+, III +/− | — |
| **左偏** | I+, III− | 左室肥厚、左前分支阻滞 |
| **右偏** | I−, III+ | 右室肥厚、肺心病 |

---

## 🩺 四、典型心电图异常

| 异常 | ECG表现 | 常见原因 |
|:-----|:--------|:---------|
| **窦性心动过速** | HR>100，P正常 | 运动/发热/甲亢/心衰 |
| **窦性心动过缓** | HR<60，P正常 | 运动员/甲减 |
| **房颤** | P波消失→f波，RR绝对不齐 | 高血压/二尖瓣病/甲亢 |
| **室性早搏** | 宽大畸形QRS，前无P波 | 器质性心脏病/电解质紊乱 |
| **ST段抬高** | ST抬高>1mm | **急性心梗** |
| **ST段压低** | ST下斜/水平压低 | **心肌缺血** |
| **T波倒置** | T与QRS方向相反 | 心肌缺血/心肌病 |
| **长QT** | QTc>0.44s | 药物/电解质/遗传 |
| **一度AVB** | PR>0.20s | 房室传导延迟 |
| **三度AVB** | P与QRS完全分离 | 严重传导系统病变 |

---

## ❗ 易混点

- 🚨 ECG只反映**电活动**——有心电≠有收缩（电机被分离时ECG正常但无脉搏）
- 🚨 **PR间期反映房室结传导**，QRS宽度反映**室内传导**
- 🚨 胸导联V₁~V₆：前三看右心/间隔，后三看左心
- 🚨 ST抬高 = **急性透壁心梗**（紧急！）
