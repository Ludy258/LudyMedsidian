# 血管内皮功能

## 📌 定义

血管内皮**不只是一层物理屏障**，更是人体最大的**旁分泌/内分泌"器官"**（总面积~1000m²，重约1.5kg）。它通过分泌多种活性物质，主动调控**血管舒缩、凝血/抗凝、炎症和血管生长**。

> 🔑 内皮功能正常 → 血管舒张+抗凝+抗炎 → 血流顺畅。内皮功能障碍 → 动脉粥样硬化的**第一步**。

---

## 🔬 一、内皮的两大面孔——健康 vs 受损

```
                   健康内皮                    受损/功能障碍内皮
                  ────────                    ────────────────
血管舒缩：       NO↑ + PGI₂↑ → 血管舒张       ET-1↑ → 血管收缩
凝血：           NO + PGI₂ + TM → 抗凝        TF + PAI-1↑ → 促凝
抗炎/促炎：      ↓黏附分子表达                  VCAM-1/ICAM-1↑ → 白细胞黏附
平滑肌：         抑制VSMC增殖                  促进VSMC增殖+迁移(→ 内膜增厚)
```

---

## 🔬 二、内皮源性舒张因子（EDRFs）

### 1. 一氧化氮（NO）——最重要的内皮信号

**合成通路**：

```
L-精氨酸 ──eNOS（内皮型一氧化氮合酶）→ L-瓜氨酸 + **NO**
    ↑                              ↑
Ca²⁺-钙调蛋白激活            剪切应力（血流摩擦）是最强生理刺激
                              ACh/缓激肽/组胺 → 受体→Ca²⁺↑ → eNOS↑
```

**NO 的作用机制**：

```
NO(气体,脂溶性→自由扩散) → 穿入平滑肌细胞
    ↓
激活可溶性鸟苷酸环化酶(sGC) → GTP → cGMP↑
    ↓
cGMP → PKG → Ca²⁺↓ + MLC磷酸酶↑ → 平滑肌松弛 → **血管舒张**
    ↓
cGMP最终被PDE5降解 → 效应终止
```

**NO 的多重保护效应**：

| 效应 | 机制 | 意义 |
|:-----|:-----|:-----|
| **血管舒张** | cGMP → Ca²⁺↓ | 调节血压和局部血流 |
| **抗血小板** | cGMP → 抑制血小板聚集+黏附 | 防止血栓形成 |
| **抗炎** | 抑制NF-κB → ↓黏附分子表达 | 阻止单核细胞黏附→内皮下（抗AS第一步） |
| **抗增殖** | 抑制VSMC增殖+迁移 | 防止血管重构 |

**NO 产生的刺激**：

| 生理刺激 | 病理/药理刺激 |
|:---------|:------------|
| **血流剪切应力**（最强） | ACh（内皮完整时→舒张） |
| 缓激肽、组胺 | 他汀类药物（上调eNOS） |
| 运动 → 剪切应力和NO | ACEi（减少缓激肽降解） |

---

### 2. 前列环素（PGI₂）

```
磷脂 ──PLA₂→ 花生四烯酸 ──COX→ PGH₂ ──PGI₂合酶→ **PGI₂**
    ↑
剪切应力/缓激肽/ACh刺激
    ↓
PGI₂ → 平滑肌IP受体 → cAMP↑ → 血管舒张 + 抗血小板
```

> 🔑 NO 和 PGI₂ **协同保护**内皮。慢性使用 COX-2 抑制剂会减少 PGI₂ → 心血管事件风险↑（罗非昔布撤市的原因）。

---

### 3. EDHF（内皮源性超极化因子）——备用通道

当 NO 通路受损（如早期动脉粥样硬化）时，EDHF 成为**主要的舒张替代通路**。通过 K⁺通道开放 → 平滑肌超极化 → 舒张。在**阻力小动脉**中 EDHF 的贡献尤为重要。

---

## 🔬 三、内皮源性收缩因子（EDCFs）——内皮素（ET）

### 内皮素-1(ET-1)——已知最强的缩血管物质

| 特征 | 内容 |
|:-----|:-----|
| **来源** | 内皮细胞（主要）+ VSMC |
| **刺激** | 低剪切应力、缺氧、AngⅡ、凝血酶、Adr |
| **受体** | **ET_A**（平滑肌→收缩）、**ET_B**（内皮→NO释放+ET清除） |
| **效价** | 缩血管强度 = **AngⅡ 的 10 倍、NE 的 100 倍** |
| **时程** | **起效慢但持续极长**（数小时~数天） |

> 🔑 ET-1 的作用是"慢而久"——不像 NO 是秒级调节的"快进快出"信号。ET-1 升高 → 长期血管收缩+促增殖 → 肺动脉高压、慢性高血压等。

## 🔬 四、内皮功能的"探测试剂"——ACh 实验

```
健康内皮：
  注射 ACh → 内皮 M₃受体激活 → Ca²⁺↑ → eNOS↑ → NO↑↑ → 平滑肌舒张 → **血管扩张**

受损内皮：
  注射 ACh → 内皮 NO 通路受损 → NO 生成不足
           → ACh 直接作用于平滑肌 M₃受体 → 收缩！→ **矛盾性血管收缩**
```

> 🔑 冠状动脉内注射 ACh 诱发血管收缩 → 诊断**内皮功能障碍**的标志。

---

## 🧠 临床推导——内皮功能障碍 → 动脉粥样硬化的第一步

```
危险因素（高血压/高血脂/吸烟/糖尿病）
    ↓
氧化应激↑ → O₂⁻(超氧阴离子)↑ → NO被快速降解(生成ONOO⁻)
    ↓
NO生物利用度↓ → 舒张功能↓ + 抗黏附功能↓ + 抗血小板功能↓
    ↓
黏附分子(VCAM-1)↑ → 单核细胞黏附→内皮下 → 泡沫细胞形成
    ↓
与此同时：ET-1↑ → 长期缩血管+促VSMC增殖
    ↓
**内皮功能障碍 → 脂纹 → 纤维斑块 → 动脉粥样硬化进展**
```

> 🔑 **"内皮功能障碍是动脉粥样硬化的最早可测事件"**——在斑块出现之前数年即可存在。他汀类药物的"多效性"之一 = 改善内皮功能（独立于降脂）。

---

## ❗ 易混点

- 🚨 **体内最强的缩血管物质是 ET-1**（不是 AngⅡ，不是 NE）。ET-1 > AngⅡ ×10 > NE ×100
- 🚨 **ACh 对内皮完整血管→舒张**，对内皮受损血管→**矛盾性收缩**（平滑肌直接 M₃ 效应）
- 🚨 NO 是**气体**（不是激素），半衰期仅数秒，作用范围极局部
- 🚨 PDE5 抑制剂（西地那非）→ 阻止 cGMP 降解 → NO 效应延长+增强 → **血管扩张**（肺高压治疗/ED治疗）
- 🚨 COX-2 抑制剂 → PGI₂↓（但不影响 TXA₂） → 心血管风险↑
