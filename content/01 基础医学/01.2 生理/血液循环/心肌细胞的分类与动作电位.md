# 心肌细胞的分类与动作电位

## 📌 心肌细胞的分类

| 分类 | 代表细胞 | 功能 | 动作电位特点 |
|:-----|:---------|:-----|:-----------|
| **工作细胞** | 心房肌、心室肌 | 收缩泵血 | 快反应细胞，0期INa，平台期 |
| **自律细胞** | 窦房结、浦肯野纤维 | 自动产生节律 | 4期自动去极化 |
| **慢反应自律细胞** | **窦房结**、房室结 | 正常起搏 | 0期ICa-L，无平台期 |
| **快反应自律细胞** | **浦肯野纤维** | 潜在/异位起搏 | 0期INa，有平台期 |

> 🔑 关键区别：**工作细胞=收缩不自主起搏**；**自律细胞=自主起搏（不收缩）**

---

## 🔬 心室肌细胞（工作细胞）的动作电位

### 五个时相

```
mV
+30 ┤    ①
    │   /\
  0 ┤  /  \____ ② 平台期（ICa-L + IK平衡）
    │ /         \
-60 ┤/           \
    │              \
-90 ┤_______________\______
    0  1  2  3     4      (时相)
```

| 时相 | 名称 | 主要离子流 | 膜电位变化 |
|:----:|:-----|:----------|:----------|
| **0期** | 快速去极 | **INa↑**（快钠通道开放） | -90mV → +30mV |
| **1期** | 快速复极初 | **Ito↑**（瞬时外向K⁺流） | +30mV → ~0mV |
| **2期** | **平台期** | **ICa-L↑**（Ca²⁺内流）≃ **IK↓**（K⁺外流） | 维持~0mV |
| **3期** | 快速复极末 | **IK↑**（K⁺外流），ICa-L↓ | ~0mV → -90mV |
| **4期** | 静息期 | Na⁺-K⁺泵 + Na⁺-Ca²⁺交换 | 维持-90mV |

### 平台期（2期）——心肌特有

> 🔑 平台期是**心肌区别于骨骼肌**的特征性电位时相！使心肌**不应期长**→不易发生强直收缩。

- **ICa-L（L型钙通道）** 开放→Ca²⁺内流→维持去极状态
- **IK** 外流逐渐增强→两者平衡→电位维持~0mV~200ms
- 进入细胞的Ca²⁺触发**钙诱导钙释放（CICR）** → 心肌收缩

### 心室肌细胞的静息电位

- **-90mV**，主要由IK1（内向整流钾通道）维持
- 接近K⁺平衡电位（EK ≈ -95mV）

## 🔬 窦房结P细胞（慢反应自律细胞）的动作电位

| 特征 | 窦房结 | 心室肌 |
|:-----|:------|:------|
| 静息电位 | 无稳定RP，最大复极电位**-60mV** | 稳定-90mV |
| **0期** | **ICa-L**（慢Ca²⁺内流）→去极慢、幅度小 | INa（快Na⁺内流）→去极快、幅度大 |
| 1期 | 不明显 | 明显（Ito） |
| **平台期** | **无** | **有**（心肌特有） |
| **4期** | **自动去极化**（If + ICa-T） | 稳定 |
| 分类 | 慢反应自律细胞 | 快反应工作细胞 |

### 窦房结4期自动去极化的离子机制

> 🔑 这是**心脏自律性的根本**！三种离子流协同：

| 离子流 | 时程 | 作用 |
|:-------|:----|:-----|
| **If**（超极化激活阳离子流） | 4期早期 | Na⁺内流→**起搏电流**，启动去极 |
| **ICa-T**（T型钙流） | 4期中后期 | Ca²⁺内流→加速去极到阈值 |
| **IK↓**（延迟整流K⁺流衰减） | 4期全程 | K⁺外流 ↓ → 去极化阻力减小 |

## 📊 快反应 vs 慢反应细胞对比表

| 对比项 | 快反应细胞 | 慢反应细胞 |
|:-------|:---------|:---------|
| **代表** | 心室肌、心房肌、浦肯野 | **窦房结**、房室结 |
| **静息电位** | -90mV（稳定） | -60mV（不稳定，自动去极） |
| **0期离子流** | **INa**（快钠） | **ICa-L**（慢钙） |
| **0期速度** | 快（200~400 V/s） | 慢（~10 V/s） |
| **平台期** | **有** | **无** |
| **4期** | 稳定 | 自动去极化 |
| **传导速度** | 快 | 慢 |
| **阻断剂** | 河豚毒(TTX)、利多卡因 | 维拉帕米、地尔硫卓 |

---

## ❗ 易混点

- 🚨 **平台期 = Ca²⁺内流 ≈ K⁺外流**才维持~0mV电位（不是单离子流）
- 🚨 窦房结0期是**Ca²⁺内流**（不是Na⁺！）→所以CCB（钙通道阻滞剂）可减慢窦房结频率
- 🚨 **If** 是窦房结的"起搏电流"，在超极化时激活（-60mV至-100mV），Na⁺内流为主
- 🚨 心室肌4期**没有**自动去极化（不会自动兴奋）
