# 血细胞比容（Hematocrit, Hct）

## 📌 定义

将一定量的血液与抗凝剂混匀后**离心**，可分为三层：
- 上层淡黄色液体：**血浆**（占全血总体积的 55%~60%）
- 下层深红色部分：**红细胞**
- 中间薄层白色不透明：**白细胞和血小板**

**血细胞比容（hematocrit, Hct）**：血细胞在血液中所占的容积百分比。

## 🔬 正常值

| 人群 | 血细胞比容 | 备注 |
|:----|:-----------|:-----|
| 成年男性 | **40%~50%** | |
| 成年女性 | **37%~48%** | |
| 新生儿 | **~55%** | 高于成人 |

## ❗ 易混点

- 🚨 白细胞和血小板仅占总容积的 **0.15%~1%**，可忽略不计，因此 **Hct 主要反映红细胞的相对浓度**
- 🚨 **"占容积大意味着数目更多"** ——红细胞占血细胞总数的 99%，所以比容直接反映红细胞数量
- 🚨 Hct 是影响**全血黏度**的最主要因素（Hct↑ → 黏度↑ → 血流阻力↑）
