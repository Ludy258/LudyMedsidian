# 血浆蛋白（Plasma Proteins）

## 📌 定义

血浆中含有多种蛋白，统称为 **血浆蛋白（plasma protein）**，是血浆中除水外含量最多的成分。

## 🔬 分类与正常值

### 分类方法

| 方法 | 分类 | 备注 |
|:----|:-----|:-----|
| **盐析法** | 白蛋白、球蛋白、纤维蛋白原 | 三类 |
| **电泳法** | 白蛋白、α₁-球蛋白、α₂-球蛋白、β-球蛋白、γ-球蛋白 | 球蛋白进一步细分 |

### 正常值

| 成分 | 正常范围 | 占比/备注 |
|:----|:---------|:----------|
| 血浆蛋白总量 | **65~85 g/L** | |
| 白蛋白（Albumin） | **40~48 g/L** | 分子量最小，数量最多 |
| 球蛋白（Globulin） | **15~30 g/L** | α₁、α₂、β、γ 四种 |
| 白蛋白/球蛋白比值（A/G） | **1.5~2.5** | 肝病时下降 |

### 来源

- **白蛋白**和大多数球蛋白：由 **肝脏** 产生
- **γ-球蛋白**（即抗体）：由 **浆细胞**（B 淋巴细胞分化而来）产生

## 🔬 血浆蛋白的六大功能

1. **形成血浆胶体渗透压** → 保持部分水于血管内（白蛋白贡献 75%~80%）
2. **与激素可逆性结合** → 维持激素在血浆中相对较长的半衰期
3. **作为载体运输** → 脂质、维生素、代谢废物等低分子物质
4. **参与血液凝固、抗凝和纤溶** 等生理过程（如纤维蛋白原）
5. **抵御病原微生物**（γ-球蛋白 = 抗体，参与体液免疫）
6. **营养功能**

## ❗ 易混点

- 🚨 **白蛋白**：分子量最小 → 数量最多 → 贡献 75%~80% 胶体渗透压 ← **不是因为含量最多，而是分子量小→分子数多**
- 🚨 **γ-球蛋白** 来自浆细胞（不是肝脏），本质是抗体
- 🚨 **肝病时**：白蛋白↓ + γ-球蛋白↑ → A/G 比值下降（<1.5）
- 🚨 **血浆蛋白与组织液蛋白的差异**：这是血浆与组织液成分最主要的区别
