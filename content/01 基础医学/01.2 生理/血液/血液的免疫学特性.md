# 血液的免疫学特性（Immunological Properties of Blood）

## 📌 定义

血液中的各种**血细胞、抗体和补体**是机体免疫系统的重要组成部分，赋予血液防御保护功能。

免疫可分为两大类：

```
免疫系统
├── 固有免疫（innate immunity）—— 非特异性，先天获得
│   ├── 效应细胞：吞噬细胞（中性粒细胞、单核/巨噬细胞）、
│   │             树突状细胞（DC）、NK 细胞等
│   └── 效应分子：补体等
│
└── 获得性免疫（acquired immunity）—— 特异性，后天产生
    ├── 体液免疫：B 细胞 → 浆细胞 → 产生抗体
    └── 细胞免疫：T 细胞 → 活化的效应淋巴细胞 + 细胞因子
```

## 🔬 固有免疫（Innate Immunity）

由遗传获得，不具有针对某一类抗原的特异性，又称**非特异性免疫（nonspecific immunity）**。

### 固有免疫细胞

| 细胞类型 | 主要功能 |
|:---------|:---------|
| **吞噬细胞**（中性粒细胞、单核巨噬细胞系统） | 吞噬和杀灭病原微生物 |
| **树突状细胞（DC）** | 抗原提呈，连接固有与获得性免疫 |
| **自然杀伤细胞（NK 细胞）** | 杀伤病毒感染细胞和肿瘤细胞 |
| **其他**（NKT 细胞、γδT 细胞、B1 细胞等） | 补充防线 |

**功能**：固有免疫是机体抵御病原微生物入侵的**第一道防线**，启动并参与获得性免疫应答。

> 🩸 **诺贝尔奖（2011年）**：
> - **Bruce Beutler** 和 **Jules Hoffmann** — "发现 Toll 样受体在**固有免疫**激活中的作用"
> - **Ralph Steinman** — "发现**树突状细胞（DC）**及其在获得性免疫调控中的作用"

## 🔬 获得性免疫（Adaptive Immunity）

个体出生后与抗原物质接触后产生，又称**特异性免疫（specific immunity）**。

### 体液免疫

- **效应细胞**：B 淋巴细胞 → 分化为**浆细胞（plasma cell）**
- **效应分子**：**抗体（antibody）** = **免疫球蛋白（immunoglobulin, Ig）**
- **作用方式**：与抗原特异性结合，通过**中和作用、激活补体、调理作用**等发挥免疫功能

### 细胞免疫

- **效应细胞**：T 淋巴细胞 → 活化的效应淋巴细胞 + 分泌细胞因子
- **应用**：CAR-T 治疗（嵌合抗原受体 T 细胞免疫治疗）在临床肿瘤治疗中具有广泛前景

## ❗ 易混点

- 🚨 **固有免疫 ≠ 低级**——它是获得性免疫的启动基础，两者协同而非替代
- 🚨 **抗体 = γ-球蛋白**，由浆细胞产生，不是由肝脏产生
- 🚨 补体是**固有免疫分子**（非特异性），抗体是**获得性免疫分子**（特异性）
- 🚨 **2011 年诺贝尔奖纠正**：Hoffmann 发现的是 **Toll 样受体在固有免疫中的作用**，不是 DC 细胞；DC 细胞的发现归 Steinman
