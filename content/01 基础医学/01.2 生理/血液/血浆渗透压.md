# 血浆渗透压（Plasma Osmotic Pressure）

## 📌 定义

**渗透（osmosis）**：当不同浓度的溶液被半透膜分隔时，低浓度侧的水分子在渗透压差驱动下通过半透膜进入高浓度侧。

**溶液渗透压（osmotic pressure）**：取决于单位体积溶液中**溶质颗粒（分子或离子）数目的多少**，与溶质的种类和颗粒大小无关。

**血浆渗透浓度**：约 **300 mmol/L**（即 300 mOsm/(kg·H₂O)），相当于 **770 kPa** 或 **5790 mmHg**。

## 🔬 晶体渗透压 vs 胶体渗透压

```
血浆渗透压（~300 mOsm/(kg·H₂O) ≈ 770 kPa）
│
├── 晶体渗透压（crystal osmotic pressure）—— 占 ~98%
│   ├── 来源：溶解于血浆中的晶体物质（Na⁺、Cl⁻、葡萄糖等）
│   ├── 80% 来自 Na⁺ 和 Cl⁻
│   └── 核心记忆："晶 98%"
│
└── 胶体渗透压（colloid osmotic pressure）—— 占 ~1.3%
    ├── 来源：血浆蛋白（主要是白蛋白）
    ├── 约 1.3 mOsm/(kg·H₂O) ≈ 3.3 kPa / 25 mmHg
    └── 75%~80% 来自白蛋白（分子量小 → 分子数量多）
```

### 生理意义对比

| 特性 | 晶体渗透压 | 胶体渗透压 |
|:----|:-----------|:-----------|
| **溶质** | 晶体物质（Na⁺, Cl⁻, 葡萄糖等） | 血浆蛋白（主要是白蛋白） |
| **能否透过毛细血管壁** | ✓ 能自由透过 | ✗ 不易透过 |
| **能否透过细胞膜** | ✗ 大部分不易透过 | ✗ 不易透过 |
| **主要作用** | 维持**细胞内、外**水平衡 | 维持**血管内、外**水平衡 |
| **临床意义** | 保持细胞正常体积 | 维持正常血浆容量 |

> 🩸 **记忆核心**："不能通透哪，就可以维持哪的渗透压"
> - 晶体物质不易透过**细胞膜** → 维持细胞内外水平衡
> - 血浆蛋白不易透过**毛细血管壁** → 维持血管内外水平衡

## ❗ 易混点

- 🚨 晶体渗透压占 ~98%，但**生理作用范围仅限于细胞膜两侧**（调节细胞体积）
- 🚨 胶体渗透压虽然仅占 ~1.3%，但在**调节血管内外水平衡**中起关键作用
- 🚨 **白蛋白**贡献胶体渗透压的 75%~80%——不是因为含量最高，而是因为**分子量小，同样质量下分子数最多**
- 🚨 **肝、肾疾病或营养不良** → 血浆蛋白↓ → 胶体渗透压↓ → 组织液滤过↑ → **组织水肿**
- 🚨 血浆渗透压 ≈ 300 mOsm/L 这个数值要记住——它是定义"等渗"的基准
