# 出血（Hemorrhage）

## 📌 定义
- 血液从血管或心腔溢出

## 🔬 病因与机制

### 一、破裂性出血
心脏或血管壁破裂所致，出血量较多。

| 原因 | 机制 | 举例 |
|:-----|:-----|:-----|
| 血管机械性损伤 | 物理性破坏血管壁 | 割伤、刺伤 |
| 血管壁/心脏病变 | 病变致管壁薄弱破裂 | 室壁瘤、主动脉瘤、动脉粥样硬化破裂 |
| 周围病变侵蚀 | 病变侵蚀血管壁 | 恶性肿瘤、结核性空洞、消化性溃疡 |
| 静脉破裂 | 静脉压力增高或壁薄破裂 | 食管下段静脉曲张破裂 |
| 毛细血管破裂 | 局部软组织损伤 | 挫伤 |

### 二、漏出性出血
微循环毛细血管和毛细血管后静脉**通透性增高**，血液通过扩大的内皮细胞间隙和受损的基底膜漏出。

| 原因 | 机制 | 举例 |
|:-----|:-----|:-----|
| **血管壁损害** | 缺氧/感染/中毒损伤血管壁 | 脑膜炎双球菌败血症、出血热、VitC缺乏 |
| **血小板减少/功能障碍** | 生成减少/破坏过多 | 再障、白血病、DIC、血小板减少性紫癜 |
| **凝血因子缺乏** | 先天缺乏/合成减少/消耗过多 | 血友病A/B、肝病、DIC |

> 🚨 **关键数值（血小板计数与出血风险）**：
> - **< 50×10⁹/L** → 轻度出血倾向（手术/创伤后出血风险↑）
> - **< 20×10⁹/L** → 自发性出血（无诱因皮肤紫癜、黏膜出血）
> - **< 5×10⁹/L** → **严重危及生命的自发性出血**（颅内出血、消化道大出血）

## 🩺 出血类型

### 按部位
```
内出血
├── 体腔积血：心包/胸腔/腹腔/关节腔积血
├── 血肿（hematoma）：脑硬膜下/皮下/腹膜后血肿
├── 瘀点（petechia）：直径 < 2mm
├── 紫癜（purpura）：直径 3~5mm
└── 瘀斑（ecchymosis）：直径 > 5mm（常超1cm）

外出血
├── 鼻出血、咯血（肺结核、支气管扩张）
├── 呕血（消化性溃疡、食管静脉曲张）
├── 便血、尿血
```
### 出血灶颜色演变

```
红蓝色（血红蛋白）
    ↓
胆红素
    ↓
棕黄色含铁血黄素（特征性颜色改变）
```

## ⚠️ 后果

| 情况 | 后果 |
|:-----|:-----|
| 缓慢少量出血 | 可自行停止（血管收缩+血小板黏集+血凝块） |
| 短时间内丧失循环血量20%~25% | **失血性休克** |
| 重要器官出血（少量） | 严重后果：心包积血→心脏压塞；脑干出血→死亡 |
| 慢性反复性出血 | **缺铁性贫血** |
