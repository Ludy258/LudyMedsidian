# 机化（Organization）

## 📌 定义
- 由新生[[肉芽组织]]取代坏死组织、血栓、炎性渗出物或异物的过程
- 最终→纤维瘢痕

## ⚙️ 过程

```
坏死组织/血栓/异物
    ↓
肉芽组织长入（毛细血管+成纤维细胞+巨噬细胞）
    ↓
吞噬清除坏死物质
    ↓
成纤维细胞产生胶原纤维
    ↓
肉芽组织成熟 → 纤维结缔组织 → 瘢痕
```

## 🩺 常见场景

| 场景 | 说明 |
|:----|:------|
| [[坏死]]灶机化 | 坏死组织被肉芽组织取代→瘢痕 |
| [[血栓形成]]（血栓）机化 | 血栓被肉芽组织取代→与管壁紧密黏着 |
| 炎性渗出物机化 | 纤维素性渗出物机化→浆膜粘连（如心包粘连） |
| 异物机化 | 包裹→纤维囊壁形成 |

## 🔄 机化 vs 其他结局

| 结局 | 区别 |
|:----|:-----|
| **溶解吸收** | 坏死物被水解酶溶解→淋巴管/血管吸收 |
| **机化** | 肉芽组织主动长入取代坏死物 |
| **包裹** | 坏死物较大→纤维组织包裹+中央钙化 |
| **再通** | 血栓机化后裂隙形成→血流部分恢复 |
