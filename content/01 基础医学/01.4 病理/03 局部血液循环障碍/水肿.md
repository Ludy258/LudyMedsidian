# 水肿（Edema）

## 📌 定义
- 组织间隙内的体液增多称为**水肿**
- 体液积聚在体腔称为**积液**（胸腔积液、心包积液、腹腔积液/腹水）

## 🔬 分类

```
按范围          按原因
├── 全身性水肿   ├── 肾性水肿
└── 局部性水肿   ├── 肝性水肿
                 ├── 心性水肿
                 ├── 营养不良性水肿
                 ├── 淋巴性水肿
                 └── 炎性水肿
```

## ⚙️ 发病机制

### 核心机制

```
┌───────────────────────────────────────────────┐
│  影响水分进出毛细血管的因素                     │
├───────────────────────────────────────────────┤
│                                               │
│  正常：毛细血管流体静压 ≈ 胶体渗透压 → 平衡    │
│                                               │
│  失衡 → 组织间液生成 > 回流 → 水肿             │
│   ├── 流体静压↑ → 水分进入组织间隙增多         │
│   ├── 胶体渗透压↓ → 水分进入组织间隙增多       │
│   └── 淋巴回流障碍 → 组织间液聚积              │
└───────────────────────────────────────────────┘
```

> 💡 **第4机制说明**：第9版及之前的教材将**微血管壁通透性增高**列为独立机制（炎症介质、缺氧、细菌毒素→内皮细胞间隙增大→血浆蛋白渗出→组织间液胶体渗透压↑→水肿）。第10版教材整合为三大机制，但**通透性增高在炎症性水肿中的核心作用不变**，跨版本考试时建议掌握此点。
> 
> 详见分类中的[[#炎性水肿]]及病理变化中渗出液与漏出液的蛋白含量区别↓

### 一、静脉流体静压增高

| 范围 | 原因 |
|:-----|:-----|
| **局部性** | 静脉回流障碍、肿瘤压迫、静脉血栓形成 |
| **全身性** | 右心充血性心力衰竭 |

```
右心衰竭
    ↓
静脉流体静压↑ + RAAS激活
    ↓
肾脏水钠潴留 → 血容量↑
    ↓
心排出量不能增加 → 静脉内液体积存 → 压力↑
    ↓
组织间液体增加 → 水肿
```

### 二、血浆胶体渗透压降低

**机制**：血浆白蛋白合成减少或大量丧失

```
血浆白蛋白↓ → 胶体渗透压↓ → 平均实际滤过压↑
    ↓
组织液生成↑ → 血浆容量↓ → 肾灌注↓ → 继发性醛固酮增多
    ↓
水钠潴留（但不能纠正低白蛋白）→ 加重水肿
```

### 三、淋巴回流障碍

**机制**：淋巴道堵塞→含蛋白水肿液在组织间隙聚积

| 举例 | 原因 |
|:-----|:-----|
| 乳腺癌手术/放疗 | 患侧上肢严重水肿 |
| 乳腺癌浸润表浅淋巴管 | "橘皮"样外观 |
| 丝虫病→腹股沟淋巴管纤维化 | 下肢和阴囊水肿→**象皮病（elephantiasis）** |

## 🩺 病理变化

| 层次 | 表现 |
|:-----|:------|
| **大体** | 组织肿胀，颜色苍白而质软，切面呈胶冻样 |
| **镜下** | 水肿液积聚于细胞和纤维结缔组织之间/腔隙内，HE染色透亮空白区 |

> 蛋白质含量高（炎症性水肿）→ 同质性微粒状深红染
> 蛋白质含量少（心性/肾性水肿）→ 淡红染

## 各部位水肿特点

### 1. 皮下水肿
- **右心衰竭**：体位性水肿（站立→下肢，卧床→骶尾部）
- **肾功能不全/肾病综合征**：早期眼睑水肿
- **特点**：皮肤紧张、苍白，指压留凹陷→**凹陷性水肿（pitting edema）**

### 2. 肺水肿
- **最常见原因**：左心室心力衰竭
- **其他**：肾衰竭、ARDS、肺部感染、过敏反应
- **大体**：肺肿胀有弹性、质实，重量↑2~3倍，切面淡红色泡沫状液体
- **影响**：阻碍氧气交换 + 有利于细菌感染

### 3. 脑水肿
- **原因**：脑脓肿/肿瘤周围、脑炎、高血压危象、脑外伤
- **大体**：脑组织肿胀，脑回变扁平，脑沟变浅，重量增加
- **影响**：颅内压↑→**脑疝**→压迫脑干血管→**快速死亡**

## ⚠️ 对机体的影响

| 部位 | 影响 |
|:-----|:-----|
| 全身性皮下水肿 | 提示心衰/肾衰竭/营养不良 |
| 局部皮肤水肿 | 影响伤口愈合和感染清除 |
| **肺水肿** | 影响通气功能，甚至死亡 |
| **脑水肿** | 颅内压增高、脑疝→快速死亡 |
| **喉头严重水肿** | 气管阻塞→**窒息死亡** |
