# 炎症介质（Inflammatory Mediators）

## 📌 定义
炎症介质是介导炎症反应的一系列化学因子，来源于**细胞**和**血浆蛋白**，通过与靶细胞表面受体结合发挥生物活性。

### 共同特点
1. 来源：**细胞**（颗粒储存/新合成）+ **血浆蛋白**（肝脏合成前体，经蛋白水解酶裂解激活）
2. 多数通过与靶细胞表面**特异性受体**结合发挥生物活性
3. 可使靶细胞产生**次级炎症介质**
4. 半衰期短，很快被**降解/灭活**

## 🔬 一、细胞释放的炎症介质

| 介质类别 | 代表物质 | 主要作用 |
|:---------|:---------|:---------|
| **血管活性胺** | 组胺、5-HT | 组胺→细动脉扩张、细静脉通透性↑；5-HT→血管收缩 |
| **花生四烯酸代谢产物** | PG、LT、LX | PGE₂→发热、疼痛；LTB₄→趋化；LTC₄/D₄/E₄→支气管痉挛、血管通透性↑ |
| **血小板激活因子（PAF）** | PAF | 激活血小板、增加血管通透性、支气管收缩（比组胺强100-10000倍） |
| **细胞因子** | TNF、IL-1、趋化性细胞因子 | 促进黏附分子表达、急性期蛋白合成、骨髓释放中性粒细胞、发热 |
| **活性氧** | O₂⁻、H₂O₂、NO | 杀伤微生物；大量释放→组织损伤 |
| **溶酶体酶** | 酸性水解酶、中性蛋白酶 | 降解细菌；中性蛋白酶→化脓性炎症组织破坏 |
| **神经肽** | P物质 | 传导疼痛、血管扩张、通透性增加 |
| **炎症小体** | NLRP3等 | 激活caspase-1 → IL-1β、IL-18成熟 → [[细胞焦亡]] |

### 按作用时间分类

| 类别 | 代表介质 | 释放方式 | 作用时相 |
|:-----|:---------|:---------|:---------|
| **预存介质（速发相）** | 组胺、5-HT、溶酶体酶 | 脱颗粒释放 | 数秒-数分钟，持续短（15-30min） |
| **新合成介质（迟发相）** | PG、LT、PAF、细胞因子、NO | 酶激活→新合成 | 数小时达高峰，持续较长 |

### 花生四烯酸（AA）代谢途径

```
花生四烯酸（AA）
    │
    ├── 环氧合酶途径（COX）→ 前列腺素 + 血栓素
    │   ├── PGI₂ → 抑制血小板聚集、扩张血管
    │   ├── TXA₂ → 血小板聚集、血管收缩
    │   ├── PGD₂ + PGE₂ → 血管扩张、水肿
    │   └── PGE₂ → 发热、疼痛敏感
    │
    └── 脂质氧合酶途径（LOX）→ 白三烯 + 脂质素
        ├── LTB₄ → 中性粒细胞趋化
        ├── LTC₄/D₄/E₄ → 支气管痉挛、静脉通透性↑
        └── 脂质素（lipoxin）→ 抑制中性粒细胞趋化（促炎症消散）
```

> 🖼️花生四烯酸代谢途径（COX/LOX分支 + NSAIDs/齐留通作用靶点）

## 🔬 二、血浆中的炎症介质

| 系统 | 激活途径 | 主要介质 | 作用 |
|:-----|:---------|:---------|:-----|
| **激肽系统** | Ⅻ因子→激肽原酶→激肽原→缓激肽 | 缓激肽 | 细动脉扩张、通透性↑、支气管收缩、**疼痛** |
| **补体系统** | 经典/替代/凝集素途径 | C3a、C5a | 扩张血管、增加通透性、趋化白细胞、杀伤细菌 |
| **凝血/纤溶系统** | Ⅻ因子激活 | 凝血酶、纤维蛋白多肽、FⅩa、纤溶酶 | 促进白细胞黏附、提高通透性、趋化 |

## 🔬 三、主要炎症介质作用小结

| 功能 | 炎症介质 |
|:-----|:---------|
| **血管扩张** | 前列腺素、NO、组胺 |
| **血管通透性↑** | 组胺、5-HT、C3a、C5a、缓激肽、LTC₄/D₄/E₄、PAF、P物质 |
| **趋化作用、白细胞渗出和激活** | TNF、IL-1、趋化性细胞因子 |
| **发热** | IL-1、TNF、前列腺素 |
| **疼痛** | 前列腺素、缓激肽、P物质 |
| **组织损伤** | 白细胞溶酶体酶、活性氧、NO |

## 💊 抗炎药物作用靶点（临床应用）

| 药物 | 靶点 | 效应 |
|:-----|:-----|:-----|
| **非甾体抗炎药（阿司匹林、吲哚美辛）** | 抑制环氧合酶（COX） | 抑制PG合成 |
| **齐留通（Zileuton）** | 抑制脂质氧合酶（LOX） | 抑制LTs合成 |
| **糖皮质激素** | 抑制磷脂酶A₂、COX-2、细胞因子基因转录 | **广谱抗炎** |

## ⚙️ 炎症反应终止机制
1. 炎症介质**半衰期短**，很快降解
2. 中性粒细胞半衰期短（数小时-2天），发生凋亡
3. 释放终止信号：**脂质素、TGF-β、IL-10**

## ❗ 易混点
- 🚨 **组胺**主要储存于肥大细胞颗粒，是**速发相**血管反应的主要介质
- 🚨 **缓激肽**是最强的**致痛**物质之一
- 🚨 **PAF**增加血管通透性能力是组胺的**100-10000倍**
- 🚨 糖皮质激素抗炎机制是**多靶点**的（磷脂酶A₂ + COX-2 + 细胞因子转录），而NSAIDs仅抑制COX
