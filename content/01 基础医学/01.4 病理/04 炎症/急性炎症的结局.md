# 急性炎症的结局

## 📌 概述
急性炎症有三种主要结局：**痊愈**、**迁延为慢性**、**蔓延扩散**。

## ⏳ 急性炎症的终止
炎症不会无限持续，有主动终止机制：
- **炎症介质衰减**：多数介质半衰期短，自行失活
- **中性粒细胞凋亡**：被巨噬细胞识别清除
- **终止信号释放**：脂质素（lipoxins）、TGF-β、IL-10 抑制炎症反应

```
        ┌─────────────┐
        │  急性炎症   │
        └──────┬──────┘
               │
    ┌─────────┼─────────┐
    ↓         ↓         ↓
┌───────┐ ┌───────┐ ┌───────────┐
│ 痊愈  │ │ 慢性  │ │ 蔓延扩散  │
│(多数) │ │ 炎症  │ │ (少数)    │
└───────┘ └───────┘ └─────┬─────┘
                          │
              ┌───────────┼───────────┐
              ↓           ↓           ↓
          局部蔓延    淋巴道蔓延    血行蔓延
              │           │
              ↓           ↓
          糜烂、溃疡    淋巴管炎
          瘘管、窦道    淋巴结炎
          空洞
```

## 🔬 一、痊愈
- **完全痊愈**：坏死范围小、组织再生能力强 → 恢复正常结构和功能
- **不完全痊愈**：坏死范围大、组织再生能力弱 → 肉芽组织修复 → 瘢痕形成

## 🔬 二、迁延为慢性炎症
- 致炎因子持续存在或免疫功能低下
- 急性→慢性转化（如急性肝炎→慢性肝炎）

## 🔬 三、蔓延扩散

### 1. 局部蔓延
| 方式 | 定义 |
|:-----|:-----|
| **糜烂（erosion）** | 黏膜浅层坏死脱落 |
| **溃疡（ulcer）** | 黏膜深层坏死脱落 |
| **瘘管（fistula）** | 连接两个空腔器官或体腔的通道 |
| **窦道（sinus）** | 深部组织通向体表的盲端通道 |
| **空洞（cavity）** | 坏死物液化排出后形成的空腔 |

### 2. 淋巴道蔓延
- 淋巴管炎（红色线条沿淋巴管走行）
- 淋巴结炎（淋巴结肿大、压痛）

### 3. 血行蔓延

| 类型 | 定义 | 特点 |
|:-----|:-----|:-----|
| **菌血症（bacteremia）** | 细菌入血，**无中毒症状** | 血液中可查到细菌；早期大叶性肺炎、流脑 |
| **毒血症（toxemia）** | 细菌毒性产物/毒素入血 | 高热、寒战、实质细胞变性坏死；**血培养查不到病原菌** |
| **败血症（septicemia）** | 细菌入血并大量繁殖产毒 | 全身中毒症状 + 皮肤黏膜出血斑点 + 脾/淋巴结肿大；血培养**可培养出**病原菌 |
| **脓毒败血症（pyemia）** | 化脓菌引起的败血症 | 败血症表现 + 多发性栓塞性脓肿（转移性脓肿） |

## ❗ 易混点

| 鉴别点 | 菌血症 | 毒血症 | 败血症 | 脓毒败血症 |
|:-------|:-------|:-------|:-------|:----------|
| 血中细菌 | + | **-** | + | + |
| 中毒症状 | - | + | + | + |
| 转移性脓肿 | - | - | - | + |

> 💡 记忆链：菌血症（菌入血无症状）→ 毒血症（毒素入血有症状）→ 败血症（菌入血+繁殖+症状）→ 脓毒败血症（败血症+脓肿）
