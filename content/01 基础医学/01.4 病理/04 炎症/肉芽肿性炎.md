# 肉芽肿性炎（Granulomatous Inflammation）

## 📌 定义
以炎症局部**巨噬细胞及其衍生细胞增生**形成境界清楚的**结节状病灶（肉芽肿）**为特征的慢性炎症，直径0.5-2.0mm。

> 巨噬细胞衍生细胞包括：**上皮样细胞**和**多核巨细胞**

## 🔬 分类

| 类型 | 病因 | 举例 |
|:-----|:-----|:-----|
| **感染性肉芽肿** | 细菌、螺旋体、真菌、寄生虫 | 结核分枝杆菌→结核病；麻风杆菌→麻风；梅毒螺旋体→梅毒；组织胞浆菌、隐球菌、血吸虫 |
| **异物性肉芽肿** | 异物刺激 | 手术缝线、石棉、铍、滑石粉、隆乳填充物、人工血管 |
| **原因不明肉芽肿** | 不明 | **结节病（sarcoidosis）** |

## 🔬 形成条件

### 感染性肉芽肿
```
病原微生物不易消化
    ↓
细胞免疫反应
    ↓
巨噬细胞吞噬 → 提呈抗原给T细胞
    ↓
T细胞激活 → 产生 IL-2、IFN-γ
    ├── IL-2 → 激活其他T细胞
    └── IFN-γ → 巨噬细胞 → 上皮样细胞 + 多核巨细胞
```

### 异物性肉芽肿
- 异物刺激长期存在（不能被吞噬消化）
- 巨噬细胞包裹异物形成结节

## 🔬 组成成分与形态特点

| 成分 | 形态特点 |
|:-----|:---------|
| **上皮样细胞** | 胞质丰富、淡粉色、颗粒状、界限不清；核圆形或长圆形、染色浅淡、1-2个小核仁 |
| **多核巨细胞** | 核数目可达几十-几百个 |
| **Langhans巨细胞** | 结核结节中；上皮样细胞融合；核排列于细胞周边呈**马蹄形/环形** |
| **异物多核巨细胞** | 较大异物/角化上皮/尿酸盐周围；核**杂乱无章**分布 |

### 结核肉芽肿（结核结节）典型结构

```
        干酪样坏死（中心）
              ↓
    放射状排列的上皮样细胞
     + Langhans巨细胞
              ↓
       大量淋巴细胞浸润
              ↓
     纤维结缔组织包绕（外层）
```

## 🩺 其他肉芽肿类型

| 类型 | 病因 | 特点 |
|:-----|:-----|:-----|
| **脂性肉芽肿** | 创伤性脂肪坏死→脂肪释出 | 巨噬细胞吞噬脂肪 |
| **伤寒肉芽肿** | 伤寒杆菌 | 巨噬细胞吞噬伤寒杆菌、淋巴细胞、红细胞、坏死碎片 |
| **风湿肉芽肿** | [[风湿病]] | 巨噬细胞吞噬纤维素样坏死物质 |

## 🩺 临床推理链

```
肉芽肿发现
    ↓
区分感染性 / 异物性 / 原因不明
    │
    ├── 感染性 → 抗酸染色、细菌培养、PCR
    │            → 结核/麻风/梅毒/真菌/寄生虫
    │
    ├── 异物性 → 询问手术史/接触史
    │            → 缝线/石棉/滑石粉
    │
    └── 原因不明 → 排除性诊断
                 → 结节病
```

## ❗ 易混点
- 🚨 肉芽肿=**特异性**慢性炎症；一般慢性炎=**非特异性**
- 🚨 **结核结节** = 干酪样坏死 + 上皮样细胞 + Langhans巨细胞 + 淋巴细胞 + 纤维包绕
- 🚨 **Langhans巨细胞**（核马蹄形/环形排列）≠ **异物巨细胞**（核杂乱分布）
- 🚨 肉芽肿≠肉芽组织（granulation tissue）→ 前者是慢性炎症的结节状病灶，后者是修复中的新生毛细血管+成纤维细胞
