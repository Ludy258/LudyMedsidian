# 炎症的基本病理变化

## 📌 定义
炎症的基本病理变化包括 **变质（alteration）、渗出（exudation）、增生（proliferation）** 三大过程。

```
        ┌─────────┐
        │  变质   │ ← 损伤性过程（早期为主）
        │(alteration)│
        └────┬────┘
             │
    ┌────────┴────────┐
    ↓                 ↓
┌─────────┐     ┌─────────┐
│  渗出    │     │  增生   │
│(exudation)│    │(proliferation)│
└─────────┘     └─────────┘
 抗损伤+修复过程（后期为主）
```

## 🔬 1. 变质（Alteration）
- **定义**：炎症局部组织发生的**变性**和**坏死**
- **实质细胞**：[[细胞水肿]]、[[脂肪变|脂肪变性]]、[[凝固性坏死]]、[[液化性坏死]]
- **间质细胞**：[[黏液样变|黏液样变性]]、[[纤维蛋白样坏死]]
- **原因**：致病因子直接作用 + 血液循环障碍 + 炎症反应产物间接作用

## 🔬 2. 渗出（Exudation）
- **定义**：血管内的液体、蛋白质、炎症细胞通过血管壁进入组织间隙/体腔/体表/黏膜表面

### 渗出液 vs 漏出液鉴别

| 鉴别点 | 渗出液（Exudate） | 漏出液（Transudate） |
|:-------|:-----------------|:--------------------|
| **原因** | 炎症 | 非炎症 |
| **外观** | 浑浊 | 清亮 |
| **比重** | ≥1.018（多数>1.020） | <1.018 |
| **蛋白量** | >30g/L | <30g/L |
| **细胞数** | 通常>500×10⁶/L | 通常<100×10⁶/L |
| **凝固性** | 易自凝 | 不自凝 |

### 渗出液的积极作用
1. 稀释和中和毒素
2. 为白细胞带来营养、运走代谢产物
3. 抗体和补体消灭病原体
4. 纤维蛋白网限制病原扩散 + 修复支架
5. 白细胞吞噬杀灭病原、清除坏死组织
6. 刺激免疫产生

### 渗出液过多的危害
- 压迫阻塞（肺泡→换气障碍；心包/胸膜腔→压迫心/肺；喉头→窒息）
- 纤维蛋白吸收不良→机化（肺肉质变、浆膜粘连）

## 🔬 3. 增生（Proliferation）
- **实质细胞增生**：如鼻黏膜上皮和腺体、肝细胞
- **间质细胞增生**：巨噬细胞、内皮细胞、成纤维细胞
- **功能**：限制炎症扩散 + 修复损伤组织

## ❗ 易混点
- 🚨 **变质** = 损伤性过程（变性 + 坏死）；**渗出 + 增生** = 抗损伤和修复过程
- 🚨 渗出液 vs 漏出液鉴别核心：**炎症性 vs 非炎症性**
- 🚨 增生不仅见于慢性炎症，急性炎症后期也有（如急性肾小球肾炎内皮细胞增生）
