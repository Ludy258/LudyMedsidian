# 炎症的局部表现与全身反应

## 📌 概述
炎症的临床表现分为**局部表现**（红、肿、热、痛、功能障碍）和**全身反应**（发热、白细胞改变等）。

## 🔬 局部表现：红、肿、热、痛、功能障碍

| 表现 | 机制 |
|:-----|:-----|
| **红（rubor）** | 局部血管扩张、充血 |
| **肿（tumor）** | 血管通透性增高，液体和细胞渗出 |
| **热（calor）** | 动脉性充血、血流加快、代谢旺盛 |
| **痛（dolor）** | 渗出物压迫 + 炎症介质作用于感觉神经末梢 |
| **功能障碍（functio laesa）** | 上述基础上引起（如关节炎→活动不灵活） |

## 🔬 全身反应

### 1. 发热（Fever）

```
外源性致热原（细菌产物）
    ↓
刺激白细胞释放 IL-1、TNF
    ↓
作用于下丘脑体温调节中枢
    ↓
PGE₂ 合成增加
    ↓
体温调定点上调 → 发热
```

### 2. 急性期反应蛋白（Acute Phase Proteins）

炎症刺激（IL-6、IL-1、TNF）→ 肝细胞合成急性期反应蛋白↑

| 蛋白 | 变化 | 临床意义 |
|:-----|:-----|:---------|
| **C反应蛋白（CRP）** | 急剧↑（可达1000倍） | 临床最常用的炎症标志物 |
| **血清淀粉样蛋白A（SAA）** | 急剧↑ | 病毒性感染时↑可能早于CRP |
| **纤维蛋白原** | ↑ | 导致ESR（血沉）增快 |
| **铁蛋白** | ↑ | 炎症性贫血相关 |
| **补体蛋白** | ↑ | 增强免疫防御 |
| **白蛋白** | **↓**（负性急性期蛋白） | 与炎症程度呈负相关 |

> 💡 **CRP**是临床最实用的炎症指标：细菌感染时显著↑，病毒感染时轻度↑或不↑，可用于鉴别

### 3. 末梢血白细胞计数改变

| 感染类型 | 白细胞变化 | 代表性疾病 |
|:---------|:-----------|:-----------|
| **细菌感染** | **中性粒细胞↑**（可达15-20×10⁹/L） | 类白血病反应时可达40-100×10⁹/L |
| **寄生虫感染/过敏** | **嗜酸性粒细胞↑** | 寄生虫病、过敏反应 |
| **某些病毒感染** | **单核巨噬细胞/淋巴细胞↑** | 传染性单核细胞增多症 |
| **多数病毒/立克次体/原虫** | **白细胞减少** | 病毒性感冒、立克次体病 |

### 4. 核左移（Left Shift）
- IL-1和TNF促进白细胞从骨髓释放
- 杆状核中性粒细胞比例增加
- **提示**：严重感染，骨髓储备释放加速

### 5. 严重感染的全身后果

```
严重感染 → 败血症
    ├── 全身血管扩张、血浆外渗 → 休克
    └── 凝血系统激活 → DIC（弥散性血管内凝血）
```

## 🩺 临床要点
- **五个局部表现**中，红热提示动脉性充血（早期），肿提示渗出（进展期）
- **发热 + 白细胞↑ + 核左移** → 细菌性感染的典型三联征
- 白细胞减少可见于病毒、立克次体、某些原虫感染（如疟疾）
