# 化生（Metaplasia）

## 📌 定义
- 一种**分化成熟的上皮或间叶组织**被另一种分化成熟的上皮或间叶组织所取代
- 并非原细胞直接转变，而是**干细胞/未分化间充质细胞转分化**的结果

## 🔬 分类

> 🖼️ **鳞状化生镜下典型图**：支气管纤毛柱状上皮→鳞状上皮 ![[病理_化生_支气管鳞状上皮化生镜下.png]]

### 上皮组织化生
| 类型 | 常见部位/原因 |
|------|--------------|
| **鳞状上皮化生（鳞化）** | 吸烟→支气管纤毛柱状上皮→鳞状上皮；维生素A缺乏→泌尿上皮鳞化 |
| **肠上皮化生（肠化）** | 慢性胃炎→胃黏膜→小肠或大肠上皮 |
| **假幽门腺化生** | 胃窦、胃体部腺体→幽门腺 |
| **Barrett食管** | 反流性食管炎→食管下段鳞状上皮→柱状上皮 |
| **子宫颈糜烂** | 慢性宫颈炎→宫颈鳞状上皮→柱状上皮 |
| **肾盂移行上皮化生** | 肾结石/慢性感染→肾盂移行上皮→**鳞状上皮**（常见）或**腺上皮**（少见） |

### 间叶组织化生
| 类型 | 举例 |
|------|------|
| **骨或软骨化生** | 骨化性肌炎（成纤维细胞→成骨细胞） |

> 📝 **手写笔记精华**：
> - 化生仅发生于**同源性细胞间**（上皮↔上皮；间叶↔间叶）
> - 发育异常所致的组织异位 ≠ 化生
> - 上皮组织化生**原因消除后可恢复**；间叶组织化生**大多不可逆**
> - 神经纤维**不能**化生
> - 上皮-间质转化（EMT）→ 肿瘤转移的重要机制
> - 🚨 **肾盂鳞化**：结石/慢性感染→移行上皮鳞化→有恶变潜能，是[[肾盂鳞状细胞癌]]的主要发生路径
> - 化生总是病理性的

## 🩺 临床意义

| 有利 | 有害 |
|------|------|
| 增强局部抵御刺激能力 | 削弱黏膜自净功能 |
| | **持续存在→细胞恶变（癌前病变）** |

**癌变链条**：
- 支气管鳞化 → [[肺鳞状细胞癌]]
- 胃黏膜肠化 → [[胃腺癌]]
- Barrett食管 → [[食管腺癌]]
- 肾盂移行上皮鳞化 → [[肾盂鳞状细胞癌]]（慢性结石/感染的远期风险）

## ❗ 易混点
- 🚨 化生 ≠ 发育异常所致的组织异位
- 🚨 **仅发生于同源性细胞间**（上皮↔上皮；间叶↔间叶）
- 🚨 神经纤维**不能**化生
- 🚨 上皮组织化生**原因消除后可恢复**；间叶组织化生**大多不可逆**
- 🚨 上皮-间质转化（EMT）是肿瘤转移的重要机制
