# 转移性钙化（Metastatic Calcification）

## 📌 定义

转移性钙化是由于**全身性钙磷代谢障碍**（高钙血症/高磷血症），导致钙盐沉积在**原本正常的组织**中。与营养不良性钙化不同，被沉积的组织本身**没有坏死或损伤**。

> 🔑 核心区别：营养不良性钙化=钙盐沉积在**坏死的组织**上（局部因素，血钙正常）；转移性钙化=钙盐沉积在**正常组织**上（全身因素，血钙升高）。

---

## 🔬 病因——高钙血症的常见原因

| 病因 | 机制 | 关键改变 |
|:-----|:-----|:---------|
| **原发性甲状旁腺功能亢进** | PTH↑ → 骨吸收↑ + 肾排磷↑ + 1,25-(OH)₂D₃↑ → 肠钙吸收↑ | **最常见原因** |
| **恶性肿瘤** | 骨转移→溶骨；异位 PTHrP 分泌 | 血钙↑↑ |
| **维生素 D 中毒** | 1,25-(OH)₂D₃ ↑ → 肠钙吸收过多 + 骨吸收↑ | 高钙+高磷 |
| **慢性肾衰竭** | 排磷↓→高磷血症→继发性甲旁亢；1,25-(OH)₂D₃↓ | 高磷→钙磷乘积↑ |
| **乳碱综合征** | 大量摄入钙+可吸收碱 → 高钙血症 | 少见 |
| **结节病/结核** | 肉芽肿内巨噬细胞→1α-羟化酶→维生素D活化↑ | 高钙血症 |

---

## 🔬 好发部位

钙盐优先沉积在**碱性环境**的组织（pH 较高有利于钙盐析出）：

| 部位 | 机制 | 后果 |
|:-----|:-----|:-----|
| **胃黏膜** | 分泌 HCl→局部碱潮（H⁺离开后局部碱性） | 钙盐沉积 |
| **肺** | CO₂快速弥散→局部碱性微环境 | 肺泡壁钙化 |
| **肾小管** | 排H⁺+NH₃→管液碱性 | 肾钙质沉着症 |
| **角膜** | 暴露于空气→CO₂丢失→碱性 | 带状角膜病 |
| **血管中膜** | 平滑肌+弹性纤维 | 血管钙化（CKD患者常见） |

> 🔑 转移性钙化**不选择**骨骼和牙（这两个是生理性钙化的正常场所）。它偏偏沉积在"不该有钙"的软组织——胃、肺、肾、血管——造成功能障碍。

---

## 📊 营养不良性 vs 转移性钙化——对比表

| | 营养不良性钙化 | 转移性钙化 |
|:--|:------------|:---------|
| **本质** | 钙盐沉积于**坏死/变性组织** | 钙盐沉积于**正常组织** |
| **血钙** | **正常** | **升高** |
| **血磷** | 正常 | 可升高或正常 |
| **根本原因** | 局部组织损伤 | **全身性钙磷代谢障碍** |
| **好发部位** | 坏死灶、血栓、瘢痕、结核灶 | 胃、肺、肾、血管（碱性环境） |
| **组织前提条件** | 需先有组织坏死/损伤 | 不需组织损伤 |
| **常见病因** | 结核坏死、动脉粥样硬化斑块、血栓机化 | 甲旁亢、骨转移、VitD中毒、肾衰竭 |
| **功能影响** | 通常较小（已经是坏死组织） | 可严重影响器官功能 |

---

## 🩺 临床意义

- **肾钙质沉着症**（肾小管钙盐沉积） → 肾间质纤维化 → 慢性肾病恶化
- **肺钙化** → 弥散功能受限（罕见严重）
- **血管中膜钙化**（CKD患者） → 血管顺应性↓ → 收缩压↑、脉压↑、心血管事件↑
- **带状角膜病** → 角膜钙盐沉积呈水平带状混浊（提示高钙血症的眼科体征）

---

## ❗ 易混点

- 🚨 **转移性钙化 ≠ 肿瘤转移**——名字里的"转移"指钙从血液"转移"到组织，不是癌转移
- 🚨 转移性钙化**不选骨骼沉积**——骨是正常钙库，偏偏不去；而是沉积在胃、肺、肾等软组织
- 🚨 **血钙正常 + 坏死组织钙化 = 营养不良性**；**血钙升高 + 正常组织钙化 = 转移性**
- 🚨 慢性肾衰竭→磷排不出→ 高磷→钙磷乘积↑→ **既有转移性钙化又有营养不良性钙化**（透析患者常见）
