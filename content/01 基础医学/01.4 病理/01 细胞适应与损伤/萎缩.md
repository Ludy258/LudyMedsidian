# 萎缩（Atrophy）

## 📌 定义
- 已发育正常的细胞、组织或器官**体积缩小**
- 实质细胞体积缩小 + 可伴数量减少
- 🚨 **未曾发育或发育不全 ≠ 萎缩**

| 类型           | 机制             | 举例                         |
| ------------ | -------------- | -------------------------- |
| **生理性萎缩**    | 细胞凋亡实现数量减少     | 胸腺青春期萎缩；更年期后卵巢/子宫萎缩；老年睾丸萎缩 |
| **营养不良性萎缩**  | 蛋白质摄入↓/消耗↑/血供↓ | 恶病质（糖尿病、结核、肿瘤）；脑动脉粥样硬化→脑萎缩 |
| **压迫性萎缩**    | 长期受压→缺血        | 肿瘤推挤；肾盂积水→肾皮质压迫萎缩          |
| **失用性萎缩**    | 长期工作负荷↓        | 骨折后久卧→肌肉萎缩+骨质疏松            |
| **去神经性萎缩**   | 运动神经元/轴突损伤     | 脑/脊髓神经损伤→肌肉萎缩              |
| **内分泌性萎缩**   | 靶腺激素刺激↓        | 垂体坏死→肾上腺皮质萎缩               |
| **老化和损伤性萎缩** | 慢性炎症+凋亡        | 阿尔茨海默病→大脑萎缩                |

> 💡 多种因素可协同作用，如骨折后肌肉萎缩常同时涉及失用性+神经性+营养性

> 📝 **手写笔记精华**：
> - 萎缩 → 体积缩小 → 质地变硬 → 纤维间质增多 → 脂褐素沉着
> - 光镜：①实质细胞萎缩 ②间质反应性增生（纤维组织、胶质纤维）
> - 电镜：①细胞器形成减少 ②自噬溶酶体增多 ③脂褐素形成↑

## ⚙️ 机制
血供减少 → 营养匮乏 → 细胞器功能下降 → 神经内分泌失调

## 🩺 病理变化

| 层次     | 表现                             |
| ------ | ------------------------------ |
| **肉眼** | 体积↓、重量↓、颜色变褐、质地变韧、边缘变锐、包膜增厚、皱缩 |
| **光镜** | 实质细胞体积↓/数量↓；脂褐素沉着↑             |
| **电镜** | 细胞器↓；残存小体↑；自噬溶酶体↑              |
| **结局** | **可复性**（病因去除后）；严重时细胞消失→纤维组织取代  |

> 💡 **脂褐素**：萎缩细胞的标志性色素，尤见于心肌和肝细胞，称为"消耗性色素"

## 🔬 实质萎缩→间质增生（普适规律）

实质（功能性细胞）萎缩或丢失后，间质（纤维结缔组织）**反应性增生**填补空间。这是各器官共通的病理反应：

| 器官 | 实质萎缩原因 | 间质反应 | 填充细胞 |
|:-----|:-----------|:---------|:---------|
| **肾** | 入球动脉硬化→缺血→肾单位坏死 | 间质纤维化→固缩肾 | 成纤维细胞→胶原 |
| **心** | 心肌梗死→心肌坏死 | 纤维瘢痕 | 成纤维细胞→胶原 |
| **肝** | 肝炎/酒精→肝细胞坏死 | 肝硬化 | 肝星状细胞→胶原 |
| **肺** | 肺泡破坏（COPD/矽肺） | 间质纤维化 | 成纤维细胞→胶原 |
| **骨骼肌** | 失用性萎缩 | 纤维+脂肪组织取代 | 成纤维细胞+脂肪细胞 |
| **脑** ⚠️ | 脑梗死→脑软化 | **胶质瘢痕** | 星形胶质细胞（不是成纤维细胞） |

> 🔑 脑是例外——因其无成纤维细胞，实质缺损由**星形胶质细胞**增生形成的**胶质瘢痕**填补，而非纤维化。

**临床意义**：实质萎缩+间质增生→器官质地变韧（如肝硬化、肾硬化）。当间质增生足够多时，器官体积可能不缩小甚至增大→[[假性肥大]]（外观肥大，功能减退）。

## ❗ 易混点
- 🚨 萎缩 ≠ 发育不全：前者是**已发育正常**后缩小，后者是**从未发育**
