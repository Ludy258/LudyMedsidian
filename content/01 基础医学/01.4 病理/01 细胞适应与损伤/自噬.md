# 自噬（Autophagy）

## 📌 定义
- 细胞在饥饿/能量应激状态下，将自身蛋白、细胞器和胞质包裹形成囊泡，在**溶酶体**中消化降解的过程
- 既是一种**生存机制**，过度时也可导致**细胞死亡**

## 🔬 三种形式

| 形式 | 特点 |
|:----|:----|
| **分子伴侣介导的自噬** | 分子伴侣蛋白介导直接穿过溶酶体膜 |
| **小自噬** | 溶酶体膜内陷直接包绕被吞噬物质 |
| **大自噬**（最主要） | 双层膜包绕→自噬体→与溶酶体融合→自噬溶酶体 |

## ⚙️ 过程
```
自噬前体 → 自噬体（双层膜） → 与溶酶体融合 → 自噬溶酶体 → 降解
```

**调控**：20余种自噬相关基因（ATG）参与

## 🩺 功能与疾病

| 功能 | 临床意义 |
|:----|:--------|
| 维持能量稳态 | 饥饿时促进细胞生存 |
| 清除受损细胞器 | 线粒体质量控制 |
| 降解病原菌 | 巨噬细胞Atg5缺陷→对结核易感性↑ |
| 清除异常折叠蛋白 | 障碍→神经变性疾病 |
| 肿瘤 | 早期：防御；晚期：肿瘤通过自噬存活 |

## ❗ 易混点
- 🚨 自噬是"自我消化"——既是**生存工具**，过度也可成为**死亡方式**
- 🚨 肿瘤中自噬**双向作用**：抑制早期癌变vs促进晚期肿瘤存活
- 🚨 自噬 ≠ [[凋亡]]：自噬是降解自身成分，凋亡是程序性死亡
