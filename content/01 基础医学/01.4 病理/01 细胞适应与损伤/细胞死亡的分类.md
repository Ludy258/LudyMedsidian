# 细胞死亡的分类

## 📌 概述
- **意外性细胞死亡**：大多为[[坏死]]（被动，无调控）
- **调节性细胞死亡（RCD）**：有基因和信号通路调控

## 🗺️ 分类总览

```
意外性细胞死亡（accident cell death）
  └── 坏死（Necrosis）
  
调节性细胞死亡（RCD）
  ├── 凋亡（Apoptosis）— caspase依赖，无炎症
  ├── 坏死性凋亡（Necroptosis）— 坏死形态，程序性调控
  ├── 细胞焦亡（Pyroptosis）— 炎症caspase介导，有炎症
  ├── 铁死亡（Ferroptosis）— 铁依赖+脂质过氧化
  └── 自噬（Autophagy）— 溶酶体降解自身成分
```

## 🔬 对比表

| 类型 | 机制核心 | 炎症反应 | caspase | 形态特征 |
|:----|:--------|:-------:|:-------:|:--------|
| **[[凋亡]]** | 基因调控程序性 | 无 | 凋亡caspase | 凋亡小体，膜完整 |
| **[[坏死]]** | 意外性，酶解 | **有** | 无关 | 肿胀、破裂、自溶 |
| **[[坏死性凋亡]]** | RIP1/RIP3信号 | 有 | **不依赖** | 似坏死 |
| **[[细胞焦亡]]** | 炎症caspase+GSDMD | **有** | 炎症caspase | 膜微孔，似坏死 |
| **[[铁死亡]]** | 铁+脂质过氧化 | 不定 | 不依赖 | 线粒体皱缩 |
| **[[自噬]]** | 溶酶体降解 | 无 | 不依赖 | 自噬体/溶酶体 |

## 💡 抑制方式的鉴别
| 类型 | 特异性抑制剂 |
|:----|:------------|
| 凋亡 | 凋亡抑制剂 |
| 坏死性凋亡 | **坏死抑素-1（Nec-1）** |
| 焦亡 | GSDMD抑制剂 |
| 铁死亡 | **铁螯合剂**和抗氧化剂（其他抑制剂无效）|
| 自噬 | 自噬抑制剂（如3-MA） |

## ❗ 易混点
- 🚨 凋亡 + 焦亡 + 坏死性凋亡均称为程序性死亡，但通路不同
- 🚨 铁死亡的抑制剂**不能**抑制其他类型——铁螯合剂专一
- 🚨 自噬是"自我消化"——既是生存机制（饥饿时），过度时也可导致死亡
- 🚨 坏死性凋亡 = "程序性坏死"——形态像坏死，但由程序调控
