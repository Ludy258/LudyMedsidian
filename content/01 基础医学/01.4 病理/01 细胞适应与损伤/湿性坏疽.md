# 湿性坏疽（Moist Gangrene）

## 📌 定义
- 动脉阻塞**同时伴有静脉回流障碍**→组织含水量高→腐败菌大量繁殖→**全身中毒症状重**
- 是[[坏疽]]中**最常见**的类型，**预后较差**

## 🔬 发生条件

| 条件 | 说明 |
|:----|:------|
| **血供** | 动脉阻塞 + 静脉回流障碍 |
| **部位** | 与外界相通的内脏器官（肺、肠、子宫、阑尾、胆囊） |
| **病原菌** | 腐败菌大量繁殖（产吲哚、粪臭素等→恶臭） |

## 🩺 病理特点

| 方面 | 表现 |
|:----|:------|
| **大体** | 蓝绿色/污黑色、肿胀、湿润、**恶臭** |
| **分界** | **不清**——炎症弥漫性蔓延 |
| **全身中毒** | **重**——大量毒素吸收入血→中毒性休克→死亡 |

## ⚙️ 机制

```
动脉阻塞 + 静脉回流障碍
    ↓
组织水肿 → 含水量高
    ↓
有利腐败菌大量繁殖
    ↓
细菌分解蛋白质 → 吲哚、粪臭素 → 恶臭
    ↓
毒素大量吸收入血 → 全身中毒症状严重
    ↓
分界不清 → 蔓延快
```

## 🚨 常见疾病
- 坏疽性阑尾炎
- 肠坏疽→肠穿孔→腹膜炎
- 肺坏疽→脓毒症
- 产后坏疽性子宫内膜炎
- 糖尿病足（混合型）

## ❗ 易混点
- 🚨 湿性坏疽 = **最常见**的坏疽类型
- 🚨 **恶臭**是湿性坏疽的特征性表现（= 蛋白质被腐败菌分解产生吲哚、粪臭素）
- 🚨 分界不清+中毒重→需紧急清创/抗感染
- 🚨 糖尿病足常为混合型：合并感染时→湿性坏疽
