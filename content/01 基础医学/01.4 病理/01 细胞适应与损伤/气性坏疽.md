# 气性坏疽（Gas Gangrene）

## 📌 定义
- **深达肌肉的开放性创伤**继发**产气荚膜杆菌**等厌氧菌感染→产生大量气体+毒素
- 是[[坏疽]]中**发展最快、致死率最高**的类型

## 🔬 发生条件

| 条件 | 说明 |
|:----|:------|
| **前提** | 深达肌肉的开放性创伤（战伤、车祸） |
| **病原菌** | **产气荚膜杆菌（Clostridium perfringens）**——专性厌氧菌 |
| **环境** | 缺氧的深部伤口 |

## 🩺 病理特点

| 方面 | 表现 |
|:----|:------|
| **大体** | 肿胀明显，按压有**捻发感/捻发音**（皮下气体） |
| **分界** | 不清 |
| **全身中毒** | **极重**——发展迅速，数小时至数天致命 |

## ⚙️ 机制

```
深部开放性创伤 → 组织缺氧环境
    ↓
产气荚膜杆菌（厌氧菌）侵入
    ↓
分解肌糖原 → 产生大量气体（CO₂、H₂）
    ↓
组织肿胀 → 按压捻发感 + 产生溶血毒素
    ↓
毒素入血 → 全身中毒极重
    ↓
若不紧急处理 → 致死
```

## 🚨 治疗原则
- 🚨 **紧急手术清创**——切除坏死组织，切开减压
- 高压氧治疗（抑制厌氧菌生长）
- 大剂量抗生素（青霉素等）
- 必要时截肢

## ❗ 易混点
- 🚨 气性坏疽的**捻发感**是**体检标志性体征**
- 🚨 病原菌是**产气荚膜杆菌**（厌氧菌）——不是金黄色葡萄球菌/链球菌
- 🚨 **需要紧急手术**——延误治疗可致数小时内死亡
- 🚨 单纯抗生素无效，必须清创
