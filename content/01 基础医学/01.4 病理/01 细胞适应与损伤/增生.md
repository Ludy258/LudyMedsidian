# 增生（Hyperplasia）

## 📌 定义
- 细胞有丝分裂活跃导致组织或器官内**细胞数目增多**
- 常致体积增大和功能活跃
- 机制：激素刺激过度 / 生长因子与受体过度表达 / 细胞凋亡被抑制

## 🔬 分类

| 类型            | 举例                              |
| ------------- | ------------------------------- |
| **生理性代偿性增生**  | 肝部分切除后残存肝细胞增生；高海拔→骨髓红细胞前体细胞代偿增多 |
| **生理性内分泌性增生** | 女性青春期乳房腺上皮增生；月经周期子宫内膜腺体增生       |
| **病理性代偿性增生**  | 创伤愈合中成纤维细胞和毛细血管内皮增生；慢性炎症致组织增生   |
| **病理性内分泌性增生** | 雌激素过多→子宫内膜增生过长→功能性子宫出血          |

## 🩺 病理变化
- 细胞数量增多，形态正常或稍增大
- 弥漫性或局限性增生
- **过度增生**失去控制可演变为**[[肿瘤性增生与非肿瘤性增生|肿瘤性增生]]**

## 🆚 增生 vs [[再生]]

| 项目 | 增生 | 再生 |
|:-----|:-----|:-----|
| **诱因** | 激素、生长因子等 | 组织缺损 |
| **形态特点** | 实质细胞数量增多，常伴细胞肥大，器官和组织体积增大 | 实质细胞数量不变或减少后恢复，通常不伴细胞肥大 |
| **功能特点** | 常**增强** | 以**恢复**为主，常有不同程度障碍 |
| **生物学意义** | 机体对功能需求增加的**适应性**反应 | 维持正常细胞数量和组织结构完整性 |
| **共同点** | ① 都有细胞增殖 ② 均可发生于生理或病理情况下 ③ 均受机体调控 | 同左 |

> 📝 **手写笔记精华**：
> - 增生 = 实质细胞**数量增多**（有增殖能力的细胞）
> - ⚠️ **少数细胞例外**：心肌、神经、骨骼肌细胞 → **仅能肥大，不能增生**
> - 增生与[[肥大]]可**同时发生**，且通常是对**同一刺激**的反应（如妊娠期子宫：平滑肌细胞肥大 + 数量增生）

## ❗ 易混点
- 🚨 增生 ≠ [[肥大]]：增生是数量增多，肥大是体积增大
- 🚨 **有增殖能力的细胞**才能增生（肝细胞、成纤维细胞、上皮细胞等）
- 🚨 **永久性细胞**（心肌、骨骼肌、神经细胞）→ **仅能肥大，不能增生**（成年期分裂能力有限）
- 🚨 增生与[[肥大]]常**同时发生**于同一器官（如妊娠子宫、甲状腺肿），是对同一刺激的协同反应
- 🚨 过度增生是癌前病变的基础（如子宫内膜增生过长→子宫内膜癌）
