# 干酪样坏死（Caseous Necrosis）

## 📌 定义
- **特殊类型**的凝固性坏死，见于[[结核病]]
- 彻底的凝固性坏死→组织轮廓完全消失→肉眼如**奶酪**样

## 🔬 机制
- [[结核分枝杆菌]]感染 → 细胞免疫（迟发型超敏反应）
- 巨噬细胞活化 → 肿瘤坏死因子（TNF）释放 → 局部组织完全坏死
- 坏死组织中溶酶体酶少+脂质多（结核菌细胞壁富含脂质）→ 不易液化吸收

## 🔍 病理变化

| 层次 | 表现 |
|:----|:------|
> 🖼️ **大体图**：灰白/淡黄色奶酪样坏死物 ![[病理_坏死_干酪样坏死大体图.png|679]]
> 腋窝淋巴结三个，明显增大，呈结节状，切面淋巴结结构破坏，大量组织坏死，为灰黄色松脆的干酪样物质

| **大体** | 灰白/淡黄色、细腻、松软、**奶酪样或豆腐渣样** |
| **光镜** | **无结构颗粒状红染物**（HE染色）；**不见原有组织结构轮廓残影** |

> 🖼️ **镜下图**：HE染色无结构颗粒状红染物 ![[病理_坏死_干酪样坏死镜下无结构颗粒状物.jpg|679]]

> 🚨 与凝固性坏死的核心鉴别：**干酪样坏死 = 彻底凝固→无轮廓保留**

## 🩺 临床意义
- **肺结核**——最常见的干酪样坏死
- 干酪样坏死灶可：钙化（愈合）→液化（恶化→空洞形成）→包裹（稳定）
- 🚨 **空洞形成**：干酪样坏死物经支气管排出→肺内空腔→传染源（开放型肺结核）
- 干酪样坏死物不易机化（脂质多、酶解慢）→常被包裹→纤维包裹→钙化

## 🔄 转归

```
干酪样坏死
  ├── 包裹钙化（愈合）→ 纤维包裹 + 营养不良性钙化
  ├── 液化排出（恶化）→ 空洞形成（传染性↑）
  └── 扩散 → 沿支气管/淋巴道/血行播散
```

## ❗ 易混点
- 🚨 干酪样坏死是[[凝固性坏死]]的**特殊类型**，不是另一类坏死——区别在于前者**彻底破坏轮廓**（细节辨认不出），后者**保留轮廓残影**
- 🚨 干酪样坏死 = 结核病的**特征性病理改变**，但需结合抗酸染色才能确诊
- 🚨 少量干酪样坏死可被包裹→纤维化+钙化→临床愈合；大量→空洞→传染源
- 🚨 干酪样坏死的"干酪"是肉眼描述，不是成分描述
