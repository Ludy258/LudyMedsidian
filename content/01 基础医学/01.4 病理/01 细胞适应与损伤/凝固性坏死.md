# 凝固性坏死（Coagulative Necrosis）

## 📌 定义
- 坏死组织**蛋白质变性凝固**，溶酶体酶水解作用相对较弱
- 坏死区：灰黄色、干燥、质实
- **组织结构轮廓保留**（特征性！）——虽然细胞已死亡，但组织大体轮廓仍可辨认

## 🔬 机制
```
缺血/缺氧 → 细胞内酸中毒 → 蛋白质变性凝固
  → 溶酶体酶活性被抑制（酸性环境+酶量少）
    → 坏死组织不易被液化溶解 → 保持一定时间轮廓
```

## 🩺 好发部位
- **心、肝、肾、脾**等实质器官（蛋白质含量丰富的组织）
- 常见原因：缺血缺氧、细菌毒素、化学腐蚀剂

## 🔍 病理变化

| 层次 | 表现 |
|:----|:------|
| **大体** | 坏死区灰黄/灰白，干燥质实，与正常组织分界清（早期充血出血带→暗红）|
| **光镜** | 细胞核消失（固缩→碎裂→溶解），胞质红染颗粒状；**组织结构轮廓保存**（如肾梗死→肾小球轮廓可见）|

### 经典案例：肾凝固性坏死
- 肾动脉分支栓塞→肾梗死→三角形灰白坏死区
- 镜下可见：肾小球、肾小管**轮廓仍然可见**但细胞核消失

## 🩺 临床意义
- **心肌梗死** → 凝固性坏死（心梗最早期诊断：血清CK-MB、cTn↑）
- 肝梗死、肾梗死、脾梗死均为凝固性坏死
- 🚨 凝固性坏死最终也会被酶解液化→被吸收→机化→瘢痕

## 🔄 转归
```
形成 → 中性粒细胞和巨噬细胞清除坏死碎片
  → 周边肉芽组织长入（机化）
    → 纤维瘢痕
```

## ❗ 易混点
- 🚨 干酪样坏死是凝固性坏死的**特殊类型**——都源于蛋白质变性凝固，但干酪样坏死更加**彻底**→轮廓完全消失。二者是**包含**而非**对立**关系
- 🚨 心肌梗死后CK-MB和cTn升高可作为早期诊断指标→坏死初发即可检出
- 🚨 组织结构轮廓保留是凝固性坏死的**最特征**表现
