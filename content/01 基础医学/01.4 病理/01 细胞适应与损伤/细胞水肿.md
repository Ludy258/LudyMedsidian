# 细胞水肿（Cellular Swelling）

## 📌 定义
- 细胞损伤中**最早出现**的改变
- 实质是细胞容积和胞质离子浓度调节功能下降

## ⚙️ 机制链
```
线粒体受损 → ATP↓ → Na⁺-K⁺泵↓ → Na⁺↑ → H₂O↑ → 细胞水肿
         ↓
代谢产物蓄积 → 渗透压↑ → 加重水肿
```

> 📝 **手写笔记精华**（机制速记链）：
> ```
> 线粒体受损 → ATP↓ → Na⁺-K⁺泵↓ → Na⁺↑ → H₂O↑ → 细胞水肿 → 气球样变 → 溶解坏死
> ```

## 🔍 病理变化

| 层次     | 表现                                                          |
| ------ | ----------------------------------------------------------- |
| **肉眼** | 器官体积↑、边缘圆钝、包膜紧张、切面外翻、颜色变淡                                   |
| **光镜** | 细胞肿大→胞质红染细颗粒→严重时胞质空泡→**气球样变**![[病理_损伤_细胞水肿镜下.jpg\|613]] |
| **电镜** | 线粒体肿胀，内质网扩张                                                 |

> ⚠️ **气球样变**是细胞水肿的极端形式 → 进展为**溶解坏死**

- **好发部位**：线粒体丰富、代谢旺盛的实质细胞——**肝、肾、心**

## 🔬 严重程度分级

细胞水肿是一个连续进展的过程，按程度由轻到重：

| 分级 | 名称 | 光镜表现 | 本质 |
|:----:|:-----|:---------|:-----|
| **轻度** | **颗粒样变**（颗粒变性，旧称混浊肿胀/浊肿） | 胞质内出现**细小红染颗粒** | 线粒体+内质网轻度肿胀 |
| **中度** | **水样变性**（水变性） | 胞质疏松淡染，颗粒减少，出现小空泡 | 细胞器明显扩张成小泡状 |
| **重度** | **气球样变** | 细胞膨大如气球，胞质透明清亮 | 细胞器破裂，结构严重破坏 |

> 📌 **颗粒样变 = 轻度细胞水肿**，是细胞损伤中**最轻微、最早出现**的形态学改变。红染颗粒本质是肿胀的线粒体和扩张的内质网（并非新物质蓄积）。病因消除后可完全恢复。

## ❗ 易混点
- 🚨 细胞水肿是**最早可逆损伤**，但持续加重→气球样变→溶解坏死
- 🚨 胞质红染细颗粒 = 肿胀的线粒体和内质网（不是其他物质蓄积）
