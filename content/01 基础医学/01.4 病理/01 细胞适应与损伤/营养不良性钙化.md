# 营养不良性钙化（Dystrophic Calcification）

## 📌 定义
- 钙盐沉积于**坏死/即将坏死**的组织中
- 🚨 **血钙正常**——与转移性钙化的核心区别

## 🔬 常见部位

| 部位/情况         | 说明                               |
| :------------ | :------------------------------- |
| [[结核病]]干酪样坏死灶 | 愈合标志——纤维包裹+钙化                    |
| [[血栓形成]]（血栓）  | 静脉石（phlebolith）/动脉石（arteriolith） |
| [[动脉粥样硬化]]斑块 | 斑块内钙化→管壁变硬 |
| 心脏瓣膜病         | 风湿性心瓣膜炎→瓣膜钙化                     |
| [[瘢痕组织]]      | 陈旧瘢痕内钙盐沉积                        |
| [[脂肪坏死]]      | 钙皂形成（急性胰腺炎→低钙血症）                 |

## 🔍 病理变化
- **大体**：灰白色颗粒状/团块状，沙砾感或硬石感
- **镜下**：蓝色颗粒状至片块状（HE染色）

## 🆚 两种钙化对比

| 维度     | 营养不良性钙化 | [[转移性钙化]] |
| :----- | :------ | :-------- |
| **血钙** | **正常**  | **↑**     |
| **前提** | 局部坏死/退变 | 全身钙磷代谢障碍  |
| **组织** | 已损伤的组织  | 基本正常的组织   |

> 💡 判断口诀：**坏死的组织→营养不良性；高血钙的→转移性**
