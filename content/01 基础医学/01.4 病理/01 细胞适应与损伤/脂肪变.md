# 脂肪变（Fatty Change / Steatosis）

## 📌 定义
- 甘油三酯蓄积于**非脂肪细胞**的胞质内
- 最常见于**肝脏**

## 🔬 病因
- 严重感染、贫血
- 化学物中毒（磷、砷、氯仿、酒精）
- 缺氧、营养不良、糖尿病、肥胖

## ⚙️ 机制
1. 肝细胞胞质内脂肪酸**增多**
2. 甘油三酯**合成过多**
3. 脂蛋白/载脂蛋白**减少**

## 🔍 病理变化

| 层次     | 表现                                |
| ------ | --------------------------------- |
| **大体** | 肿大、色黄、质软、有油腻感 ![[病理_变性_脂肪变肝脏大体镜下.jpg\|470]] |
| **光镜** | HE→胞质内大小不等空泡；苏丹Ⅲ/Ⅳ→桔红色；锇酸→黑色      |
| **电镜** | 有界膜包绕的圆形均质小体（脂滴）                  |

> 📝 **手写笔记精华**：
> - 光镜口诀：胞质**空泡**、组织**肉肿**、质**软**、色**黄油**
> - 电镜特征：**界膜包绕**的圆形均质小体（脂滴）
> - 虎斑心：左心室内膜下和**乳头肌**部位（不是全心肌）。酒精/缺氧→心肌细胞脂肪变→呈黄色，与未受损的正常心肌暗红色相间→形成黄红色斑纹
> - 肾脂肪变性：**肾近曲小管**细胞基部

## 🩺 好发部位

| 部位          | 特点                                                 |
| ----------- | -------------------------------------------------- |
| **肝脏（最常见）** | 慢性肝淤血早期→小叶中央区先受累；严重→[[脂肪肝]]→肝硬化                    |
| **心肌**      | 慢性酒精中毒或缺氧→**虎斑心**（黄红相间，左心室内膜下和乳头肌） |
| **肾**       | 肾近曲小管细胞基部                                          |

> 🚨 **[[心肌脂肪浸润]]（脂肪心）** ≠ 心肌细胞脂肪变：前者是心外膜脂肪组织**伸入**肌层压迫心肌萎缩，并非心肌细胞自身变性。好发右心室心尖区，重度可致猝死

## ❗ 易混点
- 🚨 脂肪变是**非脂肪细胞**内甘油三酯蓄积，不是脂肪组织增多
- 🚨 心肌脂肪浸润 ≠ [[脂肪变]]（本质不同：一个外来侵入性假性肥大，一个自身变性）
- 🚨 心肌脂肪浸润→心肌萎缩→属[[假性肥大]]范畴（外观增大，实质萎缩）
- 🚨 磷中毒→**小叶周边带**肝细胞受累；慢性淤血→**小叶中央区**先受累
