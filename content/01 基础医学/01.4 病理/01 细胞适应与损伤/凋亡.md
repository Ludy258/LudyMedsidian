# 凋亡（Apoptosis）

## 📌 定义
- 活体内局部组织中单个细胞的**程序性死亡**
- 由体内外因素触发细胞内预存死亡程序而导致的**主动性**死亡方式

> 📝 **手写笔记**：凋亡 = 固缩性坏死（旧称，现已不用）

## 🔬 与坏死的鉴别（高频考点）
![[病理_凋亡_凋亡小体与坏死对比图.png]]

| 鉴别点 | 凋亡 | [[坏死]] |
|:------|:----|:--------|
| **机制** | 基因调控的程序化死亡，**主动**（自杀性） | 意外事故性死亡，**被动**（他杀性） |
| **诱因** | 生理性或轻微病理性（生长因子缺乏） | 严重缺氧/感染/中毒 |
| **范围** | 多**散在单个**细胞 | 常为**集聚多个**细胞 |
| **形态** | 细胞固缩、染色质边集、膜完整、**凋亡小体** | 细胞肿胀、膜破裂、自溶 |
| **生化** | 耗能主动、依赖ATP、**DNA梯状带**（180-200bp） | 不耗能、DNA降解不规则 |
| **炎症** | **无**炎症反应 | **有**炎症反应 |

## ⚙️ 形态学特征
1. 细胞皱缩，胞质高度嗜酸性（红染）
2. 染色质凝聚→边集→核碎裂
3. **凋亡小体形成**（特征性！）
4. 质膜完整→不引起炎症

## ⚙️ 生化特征
- **caspase**（凋亡蛋白酶）活化
- Ca²⁺/Mg²⁺-核酸内切酶活化→DNA梯状带
- 需钙蛋白酶活化

## ⚙️ 信号通路

```
外源性通路：FasL → Fas受体 → 凋亡信号
内源性通路：线粒体损伤 → 细胞色素C释放 → caspase级联
                        ↓
                 执行阶段：凋亡小体形成
```

### 相关基因
| 促进凋亡 | 抑制凋亡 | 双向 |
|:--------:|:--------:|:----:|
| BAD, BAX, BAK, TP53 | BCL2, BCLXL, BCLAL | CMYC |

## 🩺 与疾病
- **凋亡不足**：肿瘤、自身免疫病（细胞寿命延长）
- **凋亡过度**：神经变性疾病（AD、帕金森、亨廷顿）、缺血性损伤、病毒感染

## ❗ 易混点
- 🚨 **凋亡无炎症** vs **坏死有炎症**——这是最重要的临床鉴别点
- 🚨 凋亡是**主动耗能**过程，坏死是**被动不耗能**过程
- 🚨 DNA梯状带（Ladder）= 凋亡的特征性生化证据
- 🚨 凋亡小体 = 凋亡的特征性形态学证据（膜包被碎片）
