# 细胞周期调控异常

## 📌 定义

**细胞周期**是细胞从一次有丝分裂结束到下一次分裂结束的连续过程。肿瘤细胞普遍存在**细胞周期检查点失控**，导致基因组不稳定和持续增殖。

## 🔬 细胞周期概述

> 🖼️细胞周期调控机制示意（G1→S→G2→M检查点 + Cyclin-CDK调控）
> ![[病理_肿瘤_细胞周期调控机制示意.jpeg]]
> **TGF-β—转化生长因子-β；EGF—表皮生长因子；PDGF—血小板源性生长因子；DP1—转录因子DP1**

**检查点（Checkpoint）**：
- **G1/S检查点**：DNA是否损伤 → 最关键（由p53-pRb通路调控）
- **G2/M检查点**：DNA复制是否完成
- **M期检查点**：染色体是否正确排列

## 🔬 核心调控通路：pRb-E2F-Cyclin

```
生长因子信号
    ↓
Cyclin D-CDK4/6 ↑
    ↓
pRb磷酸化（失活）→ E2F释放
    ↓
E2F → 激活S期基因转录
    ↓
细胞进入S期（DNA复制）

--- 负向调控（刹车）---
TGF-β → p15/p21 ↑ → CDK活性↓ → pRb低磷酸化 → E2F被结合 → G1阻滞
p53 → p21 ↑ → CDK活性↓ → pRb低磷酸化 → E2F被结合 → G1阻滞
p16（INK4a）→ 直接抑制CDK4/6
```

## ⚙️ 肿瘤中的常见异常

| 异常环节 | 机制 | 肿瘤举例 |
|:---------|:-----|:---------|
| **Cyclin D1 过表达** | 基因扩增/易位 | 套细胞淋巴瘤、乳腺癌 |
| **CDK4 扩增** | 基因扩增 | 黑色素瘤、胶质瘤 |
| **p16（CDKN2A）缺失** | 抑癌基因失活 | 黑色素瘤、胰腺癌、肺癌 |
| **pRb 失活** | 基因突变/HPV E7蛋白结合 | 视网膜母细胞瘤、宫颈癌 |
| **p53 突变** | 丧失G1/S检查点功能 | 几乎所有肿瘤 |
| **Cyclin E 过表达** | 基因扩增 | 乳腺癌、卵巢癌 |

## 🔬 治疗相关：CDK4/6抑制剂

- **靶点**：CDK4/6 → 阻止pRb磷酸化 → 细胞周期阻滞于G1期
- **代表药物**：哌柏西利（Palbociclib）、阿贝西利（Abemaciclib）
- **适应症**：**HR阳性/HER2阴性乳腺癌**
- **敏感前提**：pRb **必须正常**（pRb缺失的肿瘤对CDK4/6抑制剂不敏感）

> 🔑 CDK4/6抑制剂依赖**功能性pRb**发挥作用 → pRb缺失的肿瘤无效

## ❗ 易混点

- 🚨 pRb失活 ≠ cyclin D1过表达：一个刹车坏了 vs 油门卡住了，结果都是细胞周期失控
- 🚨 HPV E7蛋白结合并降解pRb → 宫颈癌中pRb功能丧失
- 🚨 G1/S检查点是最重要的检查点，也是肿瘤中最常失控的
