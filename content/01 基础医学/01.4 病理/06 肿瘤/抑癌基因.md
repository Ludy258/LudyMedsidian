# 抑癌基因

## 📌 定义

**抑癌基因（Tumor Suppressor Gene）**：正常情况下抑制细胞生长、促进凋亡、参与DNA修复的基因。**两个等位基因均失活才致癌**（Knudson"二次打击"假说）。

> 🔑 癌基因 = 油门（激活→癌变），抑癌基因 = 刹车（失活→癌变）

## 🔬 主要抑癌基因

| 基因 | 染色体 | 蛋白功能 | 相关肿瘤 | 特点 |
|:-----|:-------|:---------|:---------|:-----|
| **TP53**（p53） | 17p13.1 | **"基因组卫士"**：细胞周期阻滞、DNA修复、凋亡 | **几乎所有肿瘤**（>50%） | 人类肿瘤中突变率最高的基因 |
| **RB1** | 13q14 | 细胞周期 **G1/S检查点**（结合E2F） | 视网膜母细胞瘤、骨肉瘤 | 第一个发现的抑癌基因 |
| **APC** | 5q21 | 负调控 **Wnt/β-catenin** 通路 | 家族性腺瘤性息肉病→结肠癌 | "gatekeeper"基因 |
| **BRCA1/BRCA2** | 17q21/13q12 | **DNA双链断裂修复**（同源重组） | 乳腺癌、卵巢癌 | 遗传性乳腺癌主因 |
| **PTEN** | 10q23 | 负调控 **PI3K/AKT/mTOR**通路 | 胶质母细胞瘤、前列腺癌、子宫内膜癌 | 抑癌基因中突变率仅次于p53 |
| **VHL** | 3p25 | 调控 **HIF-1α** 降解 | 肾透明细胞癌 | 缺氧通路关键 |
| **CDKN2A**（p16） | 9p21 | CDK抑制物（INK4a） | 黑色素瘤、胰腺癌 | p16也由该基因编码 |
| **NF1** | 17q11 | Ras-GAP（负调控Ras） | 神经纤维瘤病1型 | Ras通路的天然抑制 |
| **SMAD4** | 18q21 | TGF-β信号转导 | 胰腺癌、结直肠癌 | TGF-β通路关键 |
| **WT1** | 11p13 | 转录因子 | 肾母细胞瘤 | |

## ⚙️ pRb 与细胞周期调控

```
低磷酸化pRb（活性形式）→ 结合E2F → 阻断S期基因转录 → G1期阻滞
    ↓
Cyclin D-CDK4/6 → 磷酸化pRb
    ↓
高磷酸化pRb（失活）→ E2F释放 → S期基因转录 → 进入S期

p16（CDKN2A）→ 抑制CDK4/6 → pRb保持低磷酸化 → 细胞周期阻滞
```

## ❗ 易混点

- 🚨 癌基因和抑癌基因的突变方式不同：癌基因需**单等位基因**激活（显性），抑癌基因需**双等位基因**失活（隐性）
- 🚨 p53 突变 ≠ p53 缺失：突变的p53蛋白不仅失去功能，还可能获得**显性负效应**（dominant negative）
- 🚨 BRCA1/2 突变≠一定会得乳腺癌（显著升高但非100%）
- 🚨 TP53 突变在所有肿瘤中占>50%，但不同肿瘤中突变谱不同
