# 肿瘤的侵袭与转移

## 📌 定义

**侵袭（Invasion）**：肿瘤细胞突破基底膜，侵入周围组织的过程。

**转移（Metastasis）**：肿瘤细胞从原发部位侵入血管/淋巴管，到达远处器官形成**继发瘤**的过程。

> 🔑 **侵袭是转移的前提，转移是恶性肿瘤的最可靠标志。**

---

## 🔬 一、侵袭过程（三步）

```
肿瘤细胞获得侵袭能力
    ↓
① 细胞黏附改变
    ↓
② 细胞外基质降解
    ↓
③ 细胞运动迁移
```

### 1. 细胞黏附改变
- **E-钙黏蛋白（E-cadherin）↓** → 细胞间黏附减弱（EMT的标志；受[[癌基因与原癌基因|Ras/Raf/MEK/ERK通路]]及TGF-β、Wnt等信号调控）
- **整合素（Integrin）表达改变** → 细胞-基质黏附模式转换

### 2. 细胞外基质（ECM）降解
- **基质金属蛋白酶（MMPs）** ↑ → 降解基底膜（IV型胶原）和间质ECM
- **uPA系统** ↑ → 纤溶酶激活 → ECM降解

### 3. 细胞运动增强
- **自分泌运动因子（AMF）** → 细胞运动性↑
- **HGF/c-Met通路** → 促进侵袭性生长
- EMT核心特征：上皮标志↓（E-cadherin）+ 间质标志↑（Vimentin、N-cadherin）

---

## 🔬 二、转移途径

```
┌────────────────────────────────────┐
│          原发肿瘤                  │
│              ↓                    │
│      获得侵袭能力（EMT）            │
│              ↓                    │
│      侵入血管/淋巴管               │
│              ↓                    │
│        循环肿瘤细胞（CTC）          │
│              ↓                    │
│      滞留远处器官微血管             │
│              ↓                    │
│       穿出血管（MET）               │
│              ↓                    │
│       形成微转移灶 → 转移瘤         │
└────────────────────────────────────┘
```

### 1. 淋巴道转移（最常见于癌）
```
原发肿瘤 → 局部淋巴结 → 远处淋巴结 → 血行转移（晚期）
```
> 🖼️淋巴结转移癌镜下（右图显示侵入扩张淋巴管的癌细胞团）
> ![[病理_肿瘤_淋巴结转移癌大体镜下.jpeg]]

- **前哨淋巴结**：最早发生转移的淋巴结（乳腺癌、黑色素瘤分期关键）
- 淋巴结肿大不一定为转移（反应性增生可导致）

### 2. 血道转移（最常见于肉瘤）
```
原发肿瘤 → 侵入血管 → 远处器官（静脉系统）
   ↓
常见靶器官：肺 > 肝 > 骨 > 脑
```
- **门静脉系统**：胃肠道癌 → 肝转移
- **腔静脉系统**：肉瘤/肾癌 → 肺转移
- **椎静脉丛（Batson丛）**：前列腺癌/甲状腺癌 → 椎骨转移（不经肺）

### 3. 种植性转移
- 体腔内肿瘤脱落 → 种植于浆膜表面
- 典型：胃癌（尤其是**弥漫型胃癌/印戒细胞癌**，分泌大量黏液）→ **Krukenberg瘤**（**双侧卵巢**同时受累，约80%）
- 特点：多发性、广泛性

---

## 🔬 三、器官选择性（"种子与土壤"学说）

> **种子（肿瘤细胞）+ 土壤（靶器官微环境）→ 特定器官倾向性转移**

| 原发肿瘤 | 常见转移部位 |
|:---------|:-------------|
| **肺癌** | 脑、骨、肾上腺 |
| **乳腺癌** | 骨、肺、肝、脑 |
| **前列腺癌** | **骨**（成骨性转移） |
| **结直肠癌** | **肝**（门静脉） |
| **肾癌** | 肺、骨 |
| **甲状腺癌** | 骨 |
| **黑色素瘤** | 肺、脑、皮肤 |

---

## ❗ 易混点

- 🚨 **种植性转移 ≠ 扩散**：转移瘤与原发瘤组织学相同；葡萄胎播散后消退 → **不是转移**
- 🚨 **EMT ≠ MET**：EMT（上皮→间质）促进侵袭；MET（间质→上皮）促进转移灶生长
- 🚨 并非所有侵入血管的肿瘤细胞都能形成转移（绝大多数CTC被免疫清除）
- 🚨 **前哨淋巴结活检**可判断区域淋巴结转移状态，减少不必要的淋巴结清扫
