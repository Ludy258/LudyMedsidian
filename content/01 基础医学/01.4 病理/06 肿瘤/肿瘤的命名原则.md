# 肿瘤的命名原则

## 📌 通用原则

肿瘤命名 = **组织来源** + **后缀**

| 类型 | 命名公式 | 举例 |
|:-----|:---------|:-----|
| **良性上皮** | 部位+来源上皮+**瘤** | 皮肤**乳头状瘤**、结直肠**管状腺瘤** |
| **恶性上皮（癌）** | 部位+来源上皮+**癌** | **食管鳞状细胞癌**、**胃腺癌**、**膀胱移行细胞癌** |
| **良性间叶** | 部位+来源组织+**瘤** | **脂肪瘤**、**纤维瘤**、**平滑肌瘤**、**骨瘤** |
| **恶性间叶（肉瘤）** | 部位+来源组织+**肉瘤** | **脂肪肉瘤**、**纤维肉瘤**、**骨肉瘤**、**横纹肌肉瘤** |

## 🔬 特殊命名规则

### 以"母细胞瘤"结尾
| 名称 | 性质 | 说明 |
|:-----|:----|:-----|
| 视网膜母细胞瘤 | **恶性** | 儿童常见 |
| 髓母细胞瘤 | **恶性** | 小脑蚓部 |
| 肾母细胞瘤 | **恶性** | 儿童腹部肿瘤 |
| 骨母细胞瘤 | **良性** | 例外！注意区分 |
| 软骨母细胞瘤 | **良性** | 例外！ |

> 🚨 **母细胞瘤 ≠ 一定为恶性**，需具体记忆

### 以"病"或"瘤"结尾但为恶性
| 名称 | 实际性质 |
|:-----|:---------|
| **白血病** | 恶性（造血系统恶性肿瘤） |
| **黑色素瘤** | 恶性（皮肤恶性度最高的肿瘤之一） |
| **精原细胞瘤** | 恶性（睾丸） |
| **淋巴瘤** | 恶性（淋巴组织） |
| **骨髓瘤** | 恶性（浆细胞） |
| **生殖细胞瘤** | 恶性 |
| **霍奇金病** | 恶性（现已更名为霍奇金淋巴瘤） |

### 以"瘤"结尾但不是肿瘤
| 名称 | 本质 |
|:-----|:-----|
| **黄色瘤** | 脂质沉积 |
| **炎性假瘤** | 慢性炎症 |
| **动脉瘤** | 血管扩张 |
| **室壁瘤** | 心肌薄弱膨出 |
| **结核瘤** | 纤维包裹的干酪样坏死 |
| **梅毒瘤** | 梅毒性肉芽肿 |
| **错构瘤** | 正常组织排列紊乱 |
| **迷离瘤** | 正常组织异位 |

### 命名中的人名
- **Ewing 肉瘤**（尤因肉瘤）
- **Kaposi 肉瘤**
- **Hodgkin 淋巴瘤**

## ❗ 易混点

- 🚨 叫"瘤"的可能是恶性（精原细胞瘤、黑色素瘤）
- 🚨 叫"母细胞瘤"的可能是良性（骨母细胞瘤、软骨母细胞瘤）
- 🚨 不叫"癌"的可能是癌（白血病、淋巴瘤——实质是造血/淋巴组织恶性肿瘤）
- 🚨 叫"瘤"的甚至可能不是肿瘤（黄色瘤、结核瘤）
