# 良恶性肿瘤的区别

## 📌 定义

**良性肿瘤**：分化好、异型性小、无转移、生长缓慢、术后不易复发的肿瘤。

**恶性肿瘤（癌症）**：分化差、异型性大、可转移、生长快、术后易复发的肿瘤。

> 🔑 **根本区别**：分化程度（异型性大小）和**是否转移**。有无转移是判断良恶性的**金标准**。

## 🔬 对比表

| 特征 | 良性肿瘤 | 恶性肿瘤 |
|:-----|:---------|:---------|
| **分化程度** | 分化好，异型性小 | 分化差，异型性大 |
| **核分裂象** | 无或少，无病理核分裂 | 多，可见**病理性核分裂** |
| **生长速度** | 缓慢 | 较快 |
| **生长方式** | **膨胀性**或**外生性**生长（有包膜） | **浸润性**或外生性生长（无包膜） |
| **继发改变** | 少见 | 常见（出血、坏死、溃疡、感染） |
| **转移** | **不转移** | **可转移**（最关键的鉴别点） |
| **复发** | 不复发或很少复发 | 易复发 |
| **对机体影响** | 较小（压迫/阻塞为主） | 大（破坏组织、出血、感染、恶病质） |
| **大体形态** | 边界清楚，有包膜，活动度好 | 边界不清，无包膜，活动度差，质硬 |

> 🖼️恶性肿瘤细胞异型性（核大、核深染、病理性核分裂）
> ![[病理_肿瘤_恶性肿瘤细胞异型性.jpeg|612]]
> ——高度恶性的肉瘤表现出显著的细胞异型性。肿瘤细胞 核大、深染，核质比例高，细胞大小及形态差异显著（多 形性），核分裂象多，可见瘤巨细胞和异常核分裂象

## ⚠️ 例外情况（必须记住）

| 看似良性但危险 | 看似恶性但预后好 |
|:---------------|:-----------------|
| 颅内良性肿瘤（脑膜瘤）→ 压迫生命中枢可致命 | **甲状腺乳头状癌** → 预后良好 |
| | **基底细胞癌** → 局部浸润但极少转移 |

## ❗ 易混点

- 🚨 不是所有良性肿瘤都无害（位置决定危害性）
- 🚨 不是所有恶性肿瘤都致命（分化程度决定预后）
- 🚨 **转移**是恶性肿瘤最确凿的证据，但无转移 ≠ 一定是良性（如基底细胞癌原位生长）
