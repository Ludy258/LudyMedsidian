# 癌基因与原癌基因

## 📌 定义

**原癌基因（Proto-oncogene）**：正常细胞基因组中存在、参与调控细胞生长/增殖/分化的基因序列。**不致癌**，是正常生理功能所必需。

**癌基因（Oncogene）**：原癌基因发生突变或过度表达后转化而来，驱动细胞**持续增殖**。

> 🔑 原癌基因 →（突变/扩增/易位）→ 癌基因 → 细胞生长失控

## 🔬 主要癌基因及其功能

| 癌基因 | 编码蛋白 | 功能分类 | 激活方式 | 相关肿瘤 |
|:-------|:---------|:---------|:---------|:---------|
| **RAS**（K/N/H-RAS） | GTP结合蛋白（小G蛋白） | 信号转导 | **点突变** → GTP酶活性↓，持续活化 | 胰腺癌90%、结肠癌50%、肺癌 |
| **MYC** | 转录因子 | 促进细胞增殖 | **易位**、扩增 | Burkitt淋巴瘤、神经母细胞瘤 |
| **HER2/neu**（ERBB2） | 受体酪氨酸激酶 | 生长因子信号 | **扩增** | 乳腺癌20-30%、胃癌、卵巢癌 |
| **EGFR**（ERBB1） | 受体酪氨酸激酶 | 生长因子信号 | 扩增、**突变**（外显子19缺失/L858R） | 肺腺癌、胶质母细胞瘤 |
| **BCL-2** | 线粒体膜蛋白 | 抑制凋亡 | **易位** t(14;18) | 滤泡性淋巴瘤 |
| **ALK** | 受体酪氨酸激酶 | 信号转导 | **易位** → 融合基因 | 肺癌（EML4-ALK）、间变性大细胞淋巴瘤 |
| **CCND1**（Cyclin D1） | 细胞周期蛋白 | 细胞周期 | 扩增、易位 | 套细胞淋巴瘤、乳腺癌 |
| **SRC** | 非受体酪氨酸激酶 | 信号转导 | 突变 | 结肠癌 |

## 🔬 Ras-MAPK 信号通路（核心通路）

> 🖼️Ras-MAPK信号转导通路示意
> **（生长因子→Ras→Raf→MEK→ERK→转录因子→增殖）**
> ![[病理_肿瘤_RasMAPK信号转导通路.jpeg|286]]![[病理_肿瘤_癌基因信号转导通路.jpeg|381]]
> **GAP—GTP酶激活蛋白；MAPK—丝裂原激活的蛋白激酶**
> 🔑 **突变型Ras**：GTP酶活性丧失，Ras-GTP持续存在 → MAPK通路**持续激活** → 细胞持续增殖

## ⚙️ 癌基因激活的四种方式

| 方式 | 机制 | 举例 |
|:-----|:-----|:-----|
| **点突变** | 基因编码序列单个碱基改变 → 蛋白功能增强 | RAS G12V → 持续活化 |
| **基因扩增** | 基因拷贝数增加 → 蛋白过表达 | HER2扩增 → 乳腺癌 |
| **染色体易位** | 基因与强启动子并置或形成融合基因 | t(8;14) → MYC过表达 → Burkitt淋巴瘤 |
| **病毒插入** | 病毒启动子插入原癌基因附近 → 激活表达 | |

## ❗ 易混点

- 🚨 **原癌基因 ≠ 癌基因**：原癌基因是正常基因，突变后才变成癌基因
- 🚨 RAS 不是唯一通路：PI3K/AKT/mTOR 是另一条重要的下游信号通路
- 🚨 同一癌基因在不同肿瘤中激活方式可能不同（如 MYC 在 Burkitt淋巴瘤为易位，在神经母细胞瘤为扩增）
