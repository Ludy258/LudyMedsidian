# 染色体易位与融合基因

## 📌 定义

**染色体易位**：染色体片段位置交换的异常，是肿瘤中常见的基因激活方式之一。

**融合基因**：易位导致两个不同基因的部分序列拼接，形成具有致癌活性的嵌合蛋白。

## 🔬 经典易位与融合基因

| 肿瘤 | 染色体易位 | 融合基因 | 致癌机制 |
|:-----|:-----------|:---------|:---------|
| **慢性粒细胞白血病（CML）** | t(9;22)(q34;q11) | **BCR::ABL1** | 持续活化的酪氨酸激酶 → 增殖信号↑ |
| **急性早幼粒细胞白血病（APL）** | t(15;17)(q22;q21) | **PML::RARα** | 分化阻滞（维A酸靶向治疗有效） |
| **Burkitt淋巴瘤** | t(8;14)(q24;q32) | **IgH::MYC** | MYC置于IgH强启动子下 → 过表达 |
| **滤泡性淋巴瘤** | t(14;18)(q32;q21) | **IgH::BCL2** | BCL2过表达 → 凋亡抑制 |
| **Ewing肉瘤** | t(11;22)(q24;q12) | **EWS::FLI1** | 异常转录因子 |
| **间变性大细胞淋巴瘤** | t(2;5)(p23;q35) | **NPM::ALK** | 持续活化的酪氨酸激酶 |
| **肺癌（NSCLC）** | inv(2)(p21;p23) | **EML4::ALK** | 持续活化酪氨酸激酶 → 靶向药敏感 |
| **套细胞淋巴瘤** | t(11;14)(q13;q32) | **IgH::CCND1** | Cyclin D1过表达 → 细胞周期加速 |
| **黏液样脂肪肉瘤** | t(12;16)(q13;p11) | **FUS::DDIT3** | 异常转录因子 |
| **前列腺癌** | 多种易位 | **TMPRSS2::ERG** | 雄激素调控下的ERG过表达 |

## ⚙️ 分子机制

### 1. 转录激活型
```
基因A（有强启动子）↔ 基因B（有促增殖功能）
           ↓
B基因置于A启动子下 → B蛋白过表达
举例：t(8;14) → IgH启动子驱动MYC
       t(14;18) → IgH启动子驱动BCL2
```

### 2. 融合蛋白型
```
基因A（有结构域） + 基因B（有激酶结构域）
           ↓
融合蛋白：A的调控部分 + B的催化部分
           ↓
持续活化的激酶信号
举例：BCR::ABL1 → 持续酪氨酸激酶活性
      EML4::ALK → ALK激酶持续二聚化活化
```

## 🩺 临床意义

| 融合基因 | 靶向药物 | 效果 |
|:---------|:---------|:-----|
| BCR::ABL1 | **伊马替尼**（格列卫） | 开创肿瘤靶向治疗时代 |
| PML::RARα | 全反式维A酸（ATRA） | APL成为可治愈的白血病 |
| EML4::ALK | **克唑替尼**、阿来替尼 | ALK阳性NSCLC有效率高 |
| IgH::CCND1 | CDK4/6抑制剂 | 套细胞淋巴瘤新选择 |

## ❗ 易混点

- 🚨 **BCR::ABL1 = Philadelphia染色体**（Ph染色体，CML标志）
- 🚨 融合基因产物通常只在肿瘤细胞中存在 → **理想的靶向治疗靶点**
- 🚨 同一融合基因可见于不同肿瘤（ALK融合见于NSCLC和间变性大细胞淋巴瘤）
- 🚨 并非所有易位都产生融合基因（如Burkitt淋巴瘤是调控序列易位而非融合蛋白）
