# 肿瘤（Tumor/Neoplasm）

## 📌 定义

肿瘤是机体在各种致瘤因素作用下，局部组织的细胞在**基因水平上失去对生长的正常调控**，导致**克隆性异常增生**而形成的新生物（neoplasm），常表现为局部肿块。

### 肿瘤性增生 vs 非肿瘤性增生

| 特征 | 肿瘤性增生 | 非肿瘤性增生 |
|:-----|:-----------|:-------------|
| 增生性质 | **单克隆性** | 多克隆性 |
| 生长调控 | **相对自主性** | 具有自限性 |
| 分化能力 | 失去分化成熟能力 | 分化成熟 |
| 病因去除后 | **持续生长** | 停止生长 |
| 对机体影响 | 有害 | 通常有利 |

> 🔑 **肿瘤的本质**：基因水平上的克隆性异常增生，不受机体正常调控

## 🔬 良恶性肿瘤对比速览

| 维度  | 良性          | 恶性             |
| :-- | :---------- | :------------- |
| 分化  | 分化好，异型性小    | 分化差，异型性大       |
| 生长  | 缓慢，膨胀性，有包膜  | 快，浸润性，无包膜      |
| 转移  | **不转移**     | **可转移**（关键鉴别点） |
| 核分裂 | 无或极少，无病理核分裂 | 多，可见病理性核分裂     |
| 复发  | 很少          | 易复发            |
| 影响  | 局部压迫/阻塞     | 破坏、恶病质、全身影响    |

## 🔬 癌与肉瘤对比速览

| 维度 | 癌 | 肉瘤 |
|:-----|:---|:-----|
| 来源 | **上皮组织** | **间叶组织** |
| 发病率 | 80-90% | 10-20% |
| 好发年龄 | **40岁以上** | 青少年 |
| 大体 | 质硬、灰白、干燥 | 质软、灰红、**鱼肉状** |
| 镜下 | **癌巢**，间质分界清楚 | 弥漫分布 |
| 转移 | **淋巴道**为主 | **血道**为主 |

## 🩺 三大临床推理链

### 1. 肿瘤诊断推理链
```
症状/体征 → 影像学 → 肿瘤标志物 → 病理（金标准）→ 分子检测 → TNM分期 → 治疗方案
```

### 2. 肿瘤转移推理链
```
原发瘤 → EMT → 降解基底膜 → 侵入脉管 → CTC → 远处滞留 → MET → 转移瘤
```

### 3. 靶向治疗推理链
```
活检 → 基因检测 → 驱动基因 → 靶向药 → 疗效评估 → 耐药检测 → 换药
```

## 📊 TNM分期系统

**TNM分期**是国际通用的肿瘤分期标准，由T（原发肿瘤）、N（区域淋巴结）、M（远处转移）三要素组成。

### T — 原发肿瘤（Primary Tumor）

| 分级 | 定义 |
|:-----|:------|
| **Tx** | 原发肿瘤无法评估 |
| **T0** | 无原发肿瘤证据 |
| **Tis** | **原位癌**（未突破基底膜） |
| **T1-T4** | 肿瘤大小和/或局部浸润范围递增 |

### N — 区域淋巴结（Regional Lymph Nodes）

| 分级 | 定义 |
|:-----|:------|
| **Nx** | 区域淋巴结无法评估 |
| **N0** | 无区域淋巴结转移 |
| **N1-N3** | 淋巴结转移的范围和数量递增 |

### M — 远处转移（Distant Metastasis）

| 分级 | 定义 |
|:-----|:------|
| **M0** | 无远处转移 |
| **M1** | 有远处转移 |

### 临床分期（Stage Grouping）

T/N/M组合后综合为 Ⅰ-Ⅳ 期（各器官具体标准不同）：

```
Stage 0： Tis  N0  M0  → 原位癌
Stage Ⅰ：  T1  N0  M0  → 早期，局限
Stage Ⅱ：  T2-3 N0 M0 → 局部进展
Stage Ⅲ：  Tany N1-3 M0 → 区域转移
Stage Ⅳ：  Tany Nany M1 → 远处转移（晚期）
```

> 🖼️乳腺癌TNM分期示意
> ![[病理_肿瘤_乳腺癌TNM分期示意.jpeg]]
> ——N的详细说明如下。N0 ：区域淋巴结未受累或仅有孤立肿瘤细胞。N1 ：微转移；或累及同侧腋窝淋巴结1～3个；和/或临床阴性的内乳淋巴结的前哨淋巴结活检有微转移或宏转移。N2 ：累及同侧腋窝淋巴结4～9个；或影像学检查显示同侧内乳淋巴结转移但腋窝淋巴结未受累。N3 ：累及腋窝淋巴结10个或以上；或累及锁骨下淋巴结（腋窝Ⅲ水平）；或影像学检查显示同侧内乳淋巴结转移伴1个（或以上）Ⅰ、Ⅱ水平腋窝淋巴结受累；或超过3个腋窝淋巴结受累伴临床阴性的同侧内乳淋巴结的前哨淋巴结活检有微转移或宏转移；或累及同侧锁骨上淋巴结

## ❗ 关键记忆点

1. **p53是"基因组卫士"**：DNA损伤→p53活化→细胞周期阻滞/DNA修复/凋亡
2. **Ras-MAPK通路**：生长因子→Ras→Raf→MEK→ERK→增殖，Ras突变→持续活化
3. **二次打击（Knudson假说）**：抑癌基因需双等位基因失活才致癌
4. **转移三途径**：淋巴道（癌为主）、血道（肉瘤为主）、种植性
5. **免疫编辑三阶段**：清除→平衡→逃逸
6. **TNM分期**：T（原发肿瘤）、N（区域淋巴结）、M（远处转移）→ 组合后分 0-Ⅳ 期（见本笔记 TNM 章节）
7. **HPV致宫颈癌**：E6→p53降解，E7→pRb降解
