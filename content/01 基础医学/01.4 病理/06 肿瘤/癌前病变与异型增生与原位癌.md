# 癌前病变、异型增生与原位癌

## 📌 定义

**癌前病变**：具有癌变潜在可能性的良性病变，长期存在可能转变为癌。

**异型增生（Dysplasia）**：细胞增生并出现异型性，但还不足以诊断为癌。

**原位癌（Carcinoma in Situ）**：异型增生的细胞累及上皮**全层**，但尚未突破基底膜。

## 🔬 渐进过程

```
正常上皮 → 轻度异型增生 → 中度异型增生 → 重度异型增生 → 原位癌 → 浸润癌
                                                          ↓
                                                   基底膜完整    基底膜突破
```

### 异型增生分级

| 分级 | 累及范围 | 可逆性 |
|:-----|:---------|:-------|
| **轻度** | 上皮层下 **1/3** | ✅ 去除刺激后可逆转 |
| **中度** | 上皮层下 **2/3** | ⚠️ 部分可逆 |
| **重度** | 上皮层 **>2/3**，未达全层 | ❌ 难逆转，需干预 |

### 原位癌特点

- **异型细胞累及上皮全层**，但基底膜完整
- 无浸润 → **无转移**
- **临床可治愈**（切除即治愈）
- 常见于：宫颈、乳腺导管（DCIS）、皮肤（Bowen病）、膀胱（CIS）、食管

## 🔬 常见癌前病变

| 癌前病变 | 相关肿瘤 | 癌变率/风险 |
|:---------|:---------|:------------|
| **黏膜白斑** | 鳞状细胞癌 | ~5% |
| **慢性宫颈炎伴宫颈糜烂** | 宫颈癌 | 较高 |
| **乳腺纤维囊性病** | 乳腺癌 | 轻度↑ |
| **结肠腺瘤性息肉**（>2cm、绒毛状） | 结肠癌 | 较高 |
| **家族性腺瘤性息肉病** | 结肠癌 | **几乎100%** |
| **慢性萎缩性胃炎** | 胃癌 | 较低 |
| **慢性溃疡性结肠炎** | 结肠癌 | 较高（>10年） |
| **皮肤慢性溃疡/窦道** | 鳞状细胞癌 | 较低 |
| **肝硬化** | 肝细胞癌 | 较高 |
| **Barrett食管** | 食管腺癌 | 较高 |
| **宫颈HPV感染（高危型）** | 宫颈癌 | 较高 |
| **幽门螺杆菌相关胃炎** | 胃癌 | 较低 |

## ❗ 易混点

- 🚨 癌前病变 ≠ 一定会癌变（仅为风险↑，大多数不进展）
- 🚨 原位癌 ≠ 早期浸润癌（原位癌无转移能力，切除即治愈）
- 🚨 重度异型增生 ≠ 原位癌（重度异型增生未达全层，原位癌已累及全层）
- 🚨 **低级别上皮内瘤变** = 轻度-中度异型增生；**高级别上皮内瘤变** = 重度异型增生 + 原位癌
