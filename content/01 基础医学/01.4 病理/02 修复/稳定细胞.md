# 稳定细胞（Stable Cell）

## 📌 定义
- 正常情况下处于**静止期（G0）**，增殖现象不明显
- 受到**损伤刺激或激素刺激**后进入 G1 期（DNA合成前期），表现出**较强的再生能力**
- 又称**静止细胞**

## 🔬 特征

| 特征 | 说明 |
|:----|:------|
| **分裂状态** | 平时 G0 期静止；损伤/激素刺激后进入细胞周期，<1.5% 细胞处分裂期 |
| **再生能力** | ⭐⭐ 较强——条件依赖性，支架完整时可完全再生 |
| **修复方式** | 支架完整→完全[[再生]]；支架破坏→[[纤维性修复]] |

## 🧬 代表组织（高频考点）

```
腺体实质细胞：肝细胞、胰腺腺泡细胞、涎腺细胞、汗腺、皮脂腺
内分泌腺细胞：甲状腺、肾上腺、垂体等
肾小管上皮细胞
骨细胞（⚠️ 再生能力强）
软骨细胞（⚠️ 虽是稳定细胞，但再生能力很弱）
平滑肌细胞（有一定再生能力，但较弱）
神经胶质细胞（⚠️ 区别于神经元！）
```

> 📝 **易错必考点**：同为稳定细胞，**骨细胞再生能力强**，**软骨细胞再生能力很弱**——这是考试常见干扰项。

## 💡 关键例外与考点

| 要点 | 说明 |
|:----|:------|
| **肝细胞** | 稳定细胞的**典型代表**——部分切除后迅速增殖，短期内恢复原大小 |
| **神经胶质细胞** | 属于稳定细胞，可增殖→中枢损伤后形成[[胶质瘢痕]]（≠神经元，神经元是永久性细胞） |
| **肥大与增生** | 稳定细胞可同时发生[[肥大]]和[[增生]]（与永久性细胞不同，后者仅能肥大） |
| **骨 vs 软骨** | 骨细胞→再生能力强；软骨细胞→再生能力弱（缺损由纤维组织修补） |

## 🧠 三类细胞速记

| 类型 | 口诀 | 特点 |
|:----|:-----|:------|
| [[不稳定细胞]] | "天天变" | 持续分裂，再生最强 |
| **稳定细胞** | "刺激来" | G0静止，刺激后分裂 |
| [[永久性细胞]] | "坏就坏" | 出生后不分裂，不能再生 |
