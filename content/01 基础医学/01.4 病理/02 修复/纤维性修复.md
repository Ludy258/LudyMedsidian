# 纤维性修复（Fibrous Repair）

## 📌 定义
- 由**纤维结缔组织**来修复缺损，最终形成瘢痕
- 当实质细胞**再生能力不足**或**组织损伤严重**时发生
- 是[[永久性细胞]]损伤后的唯一修复方式

## 🔬 三个关键步骤

```
血管生成
    ↓
纤维化（成纤维细胞迁移增殖 + 细胞外基质积聚）
    ↓
组织重构（MMP/TIMP调控）
```

## 🩺 核心过程

```
组织损伤
    ↓
肉芽组织形成（新生毛细血管 + 成纤维细胞 + 炎症细胞）
    ↓
肉芽组织成熟 → 纤维结缔组织
    ↓
转化为瘢痕组织（大量胶原纤维 + 玻璃样变）
```

## 🆚 再生 vs 纤维性修复

| 维度 | [[再生]] | 纤维性修复 |
|------|----------|------------|
| **修复细胞** | 同种实质细胞 | 成纤维细胞 |
| **产物** | 原组织结构 | 瘢痕（胶原纤维） |
| **功能恢复** | 可完全恢复 | 仅填补缺损，功能丧失 |
| **适用条件** | 再生能力强的细胞 + 支架完整 | 再生能力弱或支架破坏 |
| **炎症参与** | 较轻 | 明显 |

> 💡 实际损伤中，**再生 + 纤维性修复常同时存在**
