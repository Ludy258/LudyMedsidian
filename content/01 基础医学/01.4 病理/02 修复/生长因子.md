# 生长因子（Growth Factor）

## 📌 概述
- 调控[[再生]]和[[纤维性修复]]的关键信号分子
- 促进细胞增殖、迁移、分化及[[细胞外基质]]合成

## 🔬 主要生长因子一览

| 生长因子 | 来源 | 主要作用 |
|----------|------|----------|
| **PDGF** | 血小板α颗粒 | 引起成纤维细胞、平滑肌细胞、单核细胞增生和游走 |
| **FGF** | 广泛 | 刺激所有间叶细胞，主要作用于内皮细胞→**促进毛细血管新生** |
| **EGF** | 下颌下腺 | 促进上皮细胞、成纤维细胞、胶质细胞及平滑肌细胞增殖 |
| **TGF-α** | 多种细胞 | 与EGF受体结合→相同作用 |
| **TGF-β** | 血小板、巨噬细胞、内皮细胞 | 低浓度→诱导PDGF合成；高浓度→抑制PDGF受体；**促进纤维化** |
| **VEGF** | 肿瘤组织等 | 促进血管生成；增加血管通透性 |
| **IL-1** | 白细胞 | 刺激成纤维细胞增殖及胶原合成 |
| **TNF** | 巨噬细胞等 | 刺激成纤维细胞增殖、胶原合成及血管再生 |

## ⚡ 关键作用链

```
组织损伤
    ↓
血小板释放 PDGF
    ↓
PDGF + 巨噬细胞分泌 TGF-β, FGF, IL-1, TNF
    ↓
成纤维细胞迁移、增殖
    ↓
TGF-β 促进胶原合成 + 抑制胶原降解
    ↓
纤维化 → 瘢痕形成

同时：
    ↓
VEGF, FGF
    ↓
内皮细胞增殖、迁移
    ↓
毛细血管新生（血管生成）
```

## 🚨 临床考点

| 考点 | 说明 |
|------|------|
| **TGF-β的双面性** | 低浓度促增殖（间接），高浓度抑增殖；是**重要的促纤维化因子** |
| **VEGF** | 肿瘤血管生成的关键靶点 |
| **PDGF** | 血小板释放→伤口愈合的起始信号 |

## 🛑 负调控

| 因素 | 作用 |
|------|------|
| **抑素（chalone）** | 组织特异性，抑制自身增殖 |
| **接触抑制** | 细胞相互接触时停止生长 |
| **血管生成抑制因子** | 血管抑素、内皮抑素、血栓黏合素1 |
