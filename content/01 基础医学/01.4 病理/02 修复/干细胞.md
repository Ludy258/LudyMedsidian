# 干细胞（Stem Cell）

## 📌 基本特点
1. 能**无限地增殖分裂**
2. 具有**处于静止状态**的能力
3. 缺少细胞系标志物
4. 可通过**非对称分裂**：一个子细胞走向分化终端，另一个保持干细胞特性

## 🔬 三大类型

| 类型 | 来源 | 分化能力 | 临床应用 |
|------|------|----------|----------|
| **胚胎干细胞（ESC）** | 囊胚内细胞群 | 全能（向三个胚层分化） | 组织移植、细胞治疗、基因治疗 |
| **成体干细胞（ASC）** | 各组织器官 | 自我更新+一定分化潜能 | 骨髓移植、组织修复 |
| **诱导性多能干细胞（iPSC）** | 已分化成体细胞重编程 | 类似ESC的全能性 | 个体化治疗、受损组织修复 |

## 🧬 成体干细胞的主要类型

| 细胞类型       | 分布           | 分化方向               |
| ---------- | ------------ | ------------------ |
| **造血干细胞**  | 骨髓、外周血       | 骨髓和血液淋巴造血细胞        |
| **间充质干细胞** | 骨髓、外周血       | 骨、软骨、腱、脂肪、肌组织、神经细胞 |
| **神经干细胞**  | 室管膜细胞、星形胶质细胞 | 神经元、星形胶质细胞、少突胶质细胞  |
| **肝脏干细胞**  | 胆管内或近胆管      | 肝细胞、胆管细胞           |
| **皮肤干细胞**  | 表皮基底层、毛囊膨大区  | 表皮、毛囊              |
| **肠上皮干细胞** | 每个隐窝周围的上皮细胞  | 潘氏细胞、刷状缘肠上皮细胞、杯状细胞 |

## 💡 临床意义

- **转分化（trans-differentiation）**：利用患者自身健康组织的干细胞，诱导分化成可替代病变组织的细胞
- 优势：克服免疫排斥 + 避免伦理问题
- 应用前景：个体化治疗、再生医学
