# 心血管系统疾病

## 📌 概述
心血管系统由**心脏**和**血管**构成。心脏是动力泵，血管是输送管道。心血管疾病是全球首位死因。

本组笔记按病种类型组织：

## 📊 核心数据速查

| 指标 | 正常值 | 病理意义 |
|:-----|:------|:---------|
| 收缩压/舒张压 | <120/<80 mmHg | ≥140/≥90 = 高血压 |
| 总胆固醇 | <5.2 mmol/L | ↑→AS风险 |
| LDL | <2.6 mmol/L | **↑→AS高风险** |
| HDL | >1.0 mmol/L | ↓→AS风险 |
| 肌钙蛋白(cTnI) | <0.04 ng/mL | ↑→心肌损伤（心梗） |
| CK-MB | <5 ng/mL | ↑→心肌损伤（再梗死指标） |
| 射血分数(EF) | 55~65% | ↓→心力衰竭 |

## 🧠 临床推理链

### 1. 胸痛鉴别推理链
```
急性胸痛
├── ECG：ST↑ → STEMI → 紧急再灌注
├── ECG：ST↓/T倒 + 肌钙蛋白↑ → NSTEMI
├── ECG：ST↓ + 肌钙蛋白正常 → 不稳定型心绞痛
├── 诱因明确+休息缓解+ECG正常 → 稳定型心绞痛
└── ECG正常+肌钙蛋白正常 → 非心源性（肺栓塞/气胸/心包炎等）
```

### 2. 高血压推理链
```
血压↑
├── 肾动脉狭窄？→ [[肾血管性高血压]]（肾素↑+血管杂音）
├── 肾实质疾病？→ 肾实质性高血压（尿异常+肾功能↓）
├── 内分泌？→ 原发性醛固酮增多症/嗜铬细胞瘤/库欣
└── 排除以上 → 原发性高血压
```
