# 支气管扩张症（Bronchiectasis）

## 📌 定义
以肺内小支气管管腔**持久性不可逆扩张**伴管壁**纤维性增厚**为特征的慢性呼吸道疾病。

> **临床表现**：慢性咳嗽、**大量脓痰**及**反复咯血**

## 🔬 病因
- **继发性（最常见）**：慢性支气管炎、麻疹和百日咳后的支气管肺炎、肺结核病等
- **先天性/遗传性**：巨大气管支气管扩张症、胰腺囊性纤维化病合并肺囊性纤维化

## ⚙️ 机制

```
反复感染（特别是化脓性炎症）
    → 管壁平滑肌、弹力纤维和软骨等支撑结构破坏
    + 支气管管壁外周肺组织慢性炎症形成的纤维瘢痕组织牵拉
    + 咳嗽时支气管腔内压增加
    → 支气管持久性不可逆扩张
```

## 🔬 病理变化
- **肉眼**：
  - 病变支气管呈**囊状或筒状扩张**
  - 可局限于一肺段或肺叶，也可累及双肺（**左肺下叶最多见**）
  - 多者肺切面呈**蜂窝状**
  - 管腔内可见黏液脓性渗出物或血性渗出物
> 🖼️扩张支气管大体![[病理_支扩_支气管扩张大体蜂窝状.png]]

- **镜下**：
  - 支气管管壁明显增厚
  - 黏膜上皮增生伴**鳞状上皮化生**
  - 黏膜下血管扩张充血，淋巴细胞、浆细胞或中性粒细胞浸润
  - 管壁腺体、平滑肌、弹力纤维和软骨**破坏、萎缩或消失**
  - 代之以肉芽组织或纤维组织
  - 邻近肺组织纤维化及淋巴组织增生

## 🩺 临床表现

| 症状 | 机制 |
|:-----|:-----|
| **频发咳嗽、大量脓痰** | 慢性炎症及化脓性炎性渗出物刺激 |
| **咯血** | 支气管管壁血管遭破坏 |
| **胸闷、闭气** | 支气管引流不畅，痰不易咳出 |
| **胸痛** | 炎症累及胸膜 |
| **气急、发绀、杵状指** | 严重肺功能障碍 |
| 肺动脉高压和[[慢性肺源性心脏病]] | 晚期并发症 |

## ⚠️ 并发症
- 肺脓肿、脓胸、脓气胸

## ❗ 易混点
- 🚨 支扩 = **持久性不可逆**扩张（区别于哮喘的可逆性痉挛）
- 🚨 左肺下叶最多见（解剖位置利于引流不畅）
- 🚨 支气管扩张症镜下特征性改变：**平滑肌、弹力纤维和软骨破坏消失**
