# 鼻咽癌（Nasopharyngeal Carcinoma）

## 📌 定义
鼻咽部上皮组织发生的恶性肿瘤。

- **地域性**：我国广东、广西、福建等地区高发（广东珠江三角洲和西江流域最高）
- **性别**：男性多于女性
- **年龄**：40～50岁
- **症状**：鼻出血、鼻塞、耳鸣、听力减退、复视、偏头痛、颈部淋巴结肿大

## 🔬 病因

| 因素 | 证据/特点 |
|:-----|:-----------|
| **EB病毒** | 瘤细胞内存在EB病毒DNA和核抗原（EBNA）；90%以上患者血清中有EB病毒抗体；**VCA-IgA阳性率可高达97%**，有诊断意义 |
| **遗传因素** | 明显地域性和家族性；高发区居民移居外地后后裔发病率仍高 |
| **化学致癌物质** | 亚硝酸胺类、多环芳烃类、微量元素镍等 |

## 🔬 病理变化

### 好发部位
鼻咽顶部 > 外侧壁和咽隐窝 > 前壁

### 大体类型
- 早期：局部黏膜粗糙或略隆起，小结节
- 进展期：结节型、菜花型、黏膜下浸润型、溃疡型
- **黏膜下浸润型**：表面黏膜尚完好，癌组织在黏膜下广泛浸润，常早期转移至颈部淋巴结
- **最常见**：结节型，其次菜花型

### 组织学类型

#### 1. 鳞状细胞癌（最常见）

| 亚型 | 特点 | 与EB病毒关系 |
|:-----|:-----|:-------------|
| **角化型（高分化鳞癌）** | 癌巢内细胞分层明显，细胞内角化，可见细胞间桥和角化珠 | — |
| **非角化型（低分化鳞癌）** | 细胞分层不明显，大小形态不一，无细胞间桥和角化珠 | **最常见类型，与EB病毒感染关系密切** |
| **未分化性鳞癌** | ①泡状核细胞癌：胞质丰富，核大空泡状，有明显核仁，淋巴细胞浸润多；**对放疗敏感** ②小细胞型 | — |

> 🖼️鼻咽鳞状细胞癌
> ![[病理_鼻咽癌_泡状核细胞癌镜下.png|679]]

#### 2. 腺癌（少见）
- 来自鼻咽黏膜柱状上皮或小腺体



## 🔬 扩散途径

```
【直接蔓延】
向上 → 破坏颅底骨质 → 颅内 → 损伤第Ⅱ～Ⅵ对脑神经
向下 → 梨状隐窝、会厌、喉上部
向外侧 → 咽鼓管 → 中耳
向前 → 鼻腔 → 眼眶；向下 → 硬腭、软腭
向后 → 上段颈椎、脊髓

【淋巴道转移】（早期常发生）
咽后壁淋巴结 → 颈上深淋巴结
    → 胸锁乳头肌后缘上1/3和2/3交界处皮下无痛性结节
    （一半以上患者以此为首发症状！）
    → 同侧为主，后期可双侧 → 融合成巨大肿块

【血行转移】（较晚）
肝、肺、骨、肾、肾上腺、胰等
```

## 🩺 预后与治疗
- 早期症状不明显，确诊时多中晚期，常有转移 → **治愈率低**
- **治疗**：以放疗为主
- 低分化鳞癌和泡状核细胞癌对放疗敏感，但易复发

## ❗ 易混点
- 🚨 **VCA-IgA阳性率97%** → 鼻咽癌重要血清学诊断指标
- 🚨 一半以上患者以**颈部淋巴结无痛性肿大**为首发症状
- 🚨 泡状核细胞癌对放疗敏感但易复发
- 🚨 黏膜下浸润型表面黏膜尚完好，易误诊
