# 慢性肺源性心脏病（Chronic Cor Pulmonale）

## 📌 定义
慢性肺疾病、肺血管及胸廓的病变引起**肺循环阻力增加**，**肺动脉压升高**而导致的**右心室壁肥厚、心腔扩大**，甚或发生**右心衰竭**的心脏病。

- **患病率**：接近0.5%
- **地区**：北方地区更常见，寒冷季节发病
- **年龄**：40岁以上，随年龄增长患病率增高

> **关键环节**：**肺动脉高压**是引起肺心病的关键环节

## 🔬 病因和发病机制

### 1. 肺疾病（最常见）

| 疾病 | 占比/特点 |
|:-----|:----------|
| **慢性阻塞性肺疾病（COPD）** | **最常见，占80%～90%**；以[[慢性支气管炎]]并发[[肺气肿]]最常见 |
| 支气管哮喘、支气管扩张症、肺尘埃沉着病、慢性纤维空洞型肺结核、肺间质纤维化 | 次之 |

```
肺毛细血管床减少 + 小血管纤维化、闭塞
    → 肺循环阻力增加
    + 阻塞性通气障碍 + 气血屏障破坏 → 气体交换面积减少
    → 肺泡气氧分压降低，二氧化碳分压升高
    → 缺氧引起肺小动脉痉挛
    + 肺血管构型改建（无肌细动脉肌化、肺小动脉中膜增生肥厚）
    → 肺循环阻力进一步增大
    → 肺动脉压升高
    → 右心肥大、扩张
```

### 2. 胸廓运动障碍性疾病（较少见）
- 严重脊柱弯曲、类风湿关节炎、胸膜广泛粘连等
- 胸廓活动受限 → 限制性通气障碍 → 肺循环阻力增加

### 3. 肺血管疾病（甚少见）
- 原发性肺动脉高压症
- 广泛或反复肺小动脉栓塞

## 🔬 病理变化

### 1. 肺部病变
除原有肺疾病表现外，主要是**肺小动脉变化**：
- **肺腺泡内小血管构型重建**：
  - 无肌型细动脉**肌化**（平滑肌增多）
  - 肌型小动脉中膜**增生、肥厚**
  - 内膜下出现纵行**平滑肌束**
- 肺小动脉炎
- 弹力纤维及胶原纤维增生
- 腔内血栓形成和机化
- 肺泡间隔毛细血管数量减少

### 2. 心脏病变（以右心室为主）
- **肉眼**：
  - 心室壁肥厚，心室腔扩张
  - 扩大的右心室占据心尖部，外观钝圆
  - 心脏重量增加（可达850g）
  - 右心室前壁肺动脉圆锥显著膨隆
  - 右心室内乳头肌和肉柱显著增粗
  - 室上嵴增厚
- **诊断标准**：肺动脉瓣下2cm处右心室前壁肌层厚度**>5mm**（正常3～4mm）
- **镜下**：
  - 右心室壁心肌细胞肥大，核增大、深染
  - 缺氧引起的心肌纤维萎缩、肌质溶解、横纹消失
  - 间质水肿和胶原纤维增生

> 🖼️右心室肥厚 >5mm 大体（心脏钝圆、肺动脉圆锥膨隆）![[病理_肺心病_右心室肥厚大于5mm大体.png]]
> ——肺气肿、肺源性心脏病：肺显著膨大，边缘钝圆，色苍白，有肺 大疱形成，右心肥大，心尖钝圆

## 🩺 临床表现

| 表现 | 特点 |
|:-----|:-----|
| **原有肺疾病症状** | 咳嗽、咳痰、喘息等 |
| **呼吸功能不全** | 呼吸困难、气急、发绀 |
| **右心衰竭** | 心悸、心率增快、全身淤血、肝脾大、下肢水肿 |
| **肺性脑病** | 严重者可因缺氧和CO₂潴留、呼吸性酸中毒 → 脑水肿 → 头痛、烦躁、抽搐、嗜睡、昏迷 |

## 🧠 临床推理链

```
COPD（最常见，80-90%）
    ↓
肺毛细血管床减少 + 小血管纤维化 + 缺氧性痉挛
    ↓
肺血管构型重建（无肌细动脉肌化、中膜增厚、平滑肌增多）
    ↓
肺循环阻力增大 → 肺动脉高压（关键环节）
    ↓
右室负荷加重 → 右室肥大、扩张
    ↓
右心衰竭
```

## 🩺 预防与治疗
- **预防**：对引发该病的肺部疾病进行早期治疗并有效控制其发展
- **关键**：右心衰竭多由急性呼吸道感染致使肺动脉压增高所诱发 → **积极治疗肺部感染是控制右心衰竭的关键**
