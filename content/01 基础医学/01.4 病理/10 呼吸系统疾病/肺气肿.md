# 肺气肿（Pulmonary Emphysema）

## 📌 定义
末梢肺组织（呼吸性细支气管、肺泡管、肺泡囊和肺泡）含气量过多伴**肺泡间隔破坏**，肺组织弹性减弱，导致肺体积膨大、通气功能降低的一种疾病状态。

- **地位**：支气管和肺部疾病最常见的并发症

## 🔬 病因
- **最常见**：[[慢性支气管炎]]
- 其他：吸烟、空气污染、肺尘埃沉着病（尘肺）

## ⚙️ 机制

```
【机制一：阻塞性通气障碍】
慢性支气管炎 → 小支气管和细支气管管壁结构破坏及纤维化增生
    → 管壁增厚、管腔狭窄 + 黏液性渗出物增多和黏液栓形成
    → 肺排气不畅，残气量过多

【机制二：呼吸性细支气管和肺泡壁弹性降低】
长期慢性炎症 → 弹力纤维破坏 → 回缩力减弱
    + 阻塞性肺通气障碍 → 细支气管和肺泡长期高张力状态
    → 弹性降低，残气量进一步增多

【机制三：α₁-抗胰蛋白酶（α₁-AT）水平降低】
炎症时氧自由基氧化α₁-AT → 失活
    → 中性粒细胞和巨噬细胞分泌的弹性蛋白酶活性增强
    → 弹力蛋白、Ⅳ型胶原和糖蛋白降解 → 肺组织结构破坏

【遗传性α₁-AT缺乏者肺气肿发病率较一般人高15倍】

【综合结果】
细支气管和肺泡腔残气量不断增多，压力升高
    → 细支气管扩张，肺泡破裂融合成含气大囊泡
    → 肺气肿形成
```

## 🔬 类型

### 1. 肺泡性肺气肿（Alveolar/Obstructive Emphysema）

| 亚型 | 特点 | 好发人群 |
|:-----|:-----|:---------|
| **腺泡中央型** | 呼吸性细支气管囊状扩张，肺泡管和肺泡囊扩张不明显 | **最常见**；中老年吸烟者、慢支患者 |
| **腺泡周围型（隔旁肺气肿）** | 呼吸性细支气管基本正常，周围肺泡管和肺泡囊扩张 | 多不合并COPD |
| **全腺泡型** | 呼吸性细支气管、肺泡管、肺泡囊和肺泡均扩张 | 青壮年、先天性α₁-AT缺乏症；严重时融合成>1cm囊泡（囊泡性肺气肿） |

### 2. 间质性肺气肿（Interstitial Emphysema）
- **原因**：肋骨骨折、胸壁穿透伤、剧烈咳嗽 → 肺内压急剧增高 → 细支气管或肺泡间隔破裂
- **特点**：空气进入肺间质，可扩散至肺门、纵隔，甚至皮下形成皮下气肿

### 3. 其他类型

| 类型 | 特点 |
|:-----|:-----|
| **瘢痕旁肺气肿** | 肺组织瘢痕灶周围，肺泡破裂融合；>2cm且破坏肺小叶间隔称**肺大疱（bullae）**，破裂可引起**气胸** |
| **代偿性肺气肿** | 肺萎缩或肺叶切除后残余肺组织代偿性过度充气，不伴气道和肺泡壁破坏 |
| **老年性肺气肿** | 老年人肺组织弹性回缩力减弱，残气量增多 |

## 🔬 病理变化
- **肉眼**：肺体积显著膨大，色灰白，边缘钝圆，柔软缺乏弹性，指压后压痕不易消退
- **镜下**：
  - 肺泡扩张，肺泡间隔变窄并断裂，相邻肺泡融合成较大囊腔
  - 肺泡间隔毛细血管床数量减少
  - 间质内肺小动脉内膜纤维性增厚
  - 小支气管和细支气管慢性炎症改变

## 🩺 临床表现

| 症状/体征 | 机制 |
|-----------|:-----|
| **呼气性呼吸困难** | 阻塞性通气障碍 |
| **气促、胸闷、发绀** | 缺氧 |
| **桶状胸（barrel chest）** | 长期过度吸气 → 肋骨上抬，肋间隙增宽，胸廓前后径加大 |
| X线：肺野扩大、横膈下降、透明度增加 | 肺容积增大 |
| 肺动脉压升高 → [[慢性肺源性心脏病]] | 肺泡间隔毛细血管床受压迫及数量减少 → 肺循环阻力增加 |

## ⚠️ 并发症

| 并发症 | 机制 |
|:-------|:-----|
| **[[慢性肺源性心脏病]]** | 肺泡间隔毛细血管床减少 + 缺氧性肺小动脉痉挛 → 肺动脉高压 → 右心衰竭 |
| **自发性气胸** | 肺大疱（>2cm）破裂 → 气体进入胸膜腔 → 突然加重的呼吸困难 |
| **呼吸衰竭** | 晚期肺泡通气量严重不足 → 低氧血症 + CO₂潴留 |
| **呼吸性酸中毒** | CO₂排出障碍 → PaCO₂↑ → pH↓ |

## 🧠 临床推理链

> **慢支 → 肺气肿 → 肺心病**（核心推理链）
