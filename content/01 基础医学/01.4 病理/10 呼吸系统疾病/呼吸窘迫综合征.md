# 呼吸窘迫综合征

## 🔬 一、成人呼吸窘迫综合征（ARDS）

### 定义
全身遭受严重创伤、感染及肺内严重疾患时出现的一种以**进行性呼吸窘迫**和**低氧血症**为特征的急性呼吸衰竭综合征。

- **别名**：休克肺、创伤后湿肺、弥漫性肺泡损伤
- **预后**：极差，病死率**50%～60%**

> **最典型病变**：**肺透明膜形成**

### 病因和发病机制

**病因**：
- 严重全身感染、创伤、休克
- 肺直接损伤：败血症、大面积烧伤、溺水、药物中毒、大量输血或输液、体外循环、透析
- 弥漫性肺感染、肺挫伤、吸入性肺炎、吸入有毒气体

```
各种损伤因素
    → 肺毛细血管和肺泡上皮严重损伤
    → 毛细血管通透性升高
    → 肺泡内及间质水肿 + 纤维蛋白大量渗出
    + Ⅱ型肺泡上皮损伤 → 肺泡表面活性物质缺失
    → 肺泡表面透明膜形成 + 肺萎陷
    → 肺内氧弥散障碍 + 通气血流比例失调
    → 低氧血症 + 呼吸窘迫
```

**详细机制**：
- 白细胞及介质（细胞因子、氧自由基、补体、花生四烯酸代谢产物）引起肺毛细血管内皮和肺泡上皮损伤
- 活化的巨噬细胞和中性粒细胞释放：氧自由基、蛋白水解酶、血管活性物质（PG、LT、TXA₂）、PAF

### 病理变化
- **肉眼**：双肺肿胀，重量增加；暗红色，湿润；散在出血点或出血斑；切面膨隆，含血量多
- **镜下**：
  - 肺间质毛细血管扩张、充血
  - 肺泡腔和肺间质内大量含蛋白质浆液（**肺水肿**）
  - **透明膜形成**：薄层红染膜状物被覆于呼吸性细支气管、肺泡管及肺泡内表面（血浆蛋白及坏死肺泡上皮碎屑）
  - 间质内点状出血和灶状坏死
  - 微血管内透明血栓和白细胞栓塞
  - 肺泡上皮弥漫性损伤
  - **数日后**：肺间质内成纤维细胞及Ⅱ型肺泡上皮大量增生，透明膜机化和胶原沉着 → **弥漫性纤维化**


## 🔬 二、新生儿呼吸窘迫综合征（NRDS）

### 定义
新生儿出生后仅出现数分钟至数小时的短暂自然呼吸便发生进行性呼吸困难、发绀等急性呼吸窘迫症状和呼吸衰竭综合征。

- **别名**：新生儿肺透明膜病（Hyaline Membrane Disease of Newborn）
- **好发**：**早产儿**、过低体重儿或过期产儿
- **特点**：有家族遗传倾向，预后差，病死率高

### 病因和发病机制

**核心问题**：**肺发育不全、缺乏肺表面活性物质**

```
胎龄22周至出生时：Ⅱ型肺泡上皮合成肺表面活性物质能力渐臻完善
    ↓
若胎儿缺氧或血液中有毒物质损伤Ⅱ型肺泡上皮
    → 板层小体减少或缺如
    → 肺表面活性物质合成和分泌障碍
    → 肺泡表面张力增加 → 肺泡膨胀不全/不扩张
    → 肺通气和换气功能障碍 → 缺氧、CO₂潴留、呼吸性酸中毒
    → 肺小血管痉挛 → 血流灌注不足
    → 肺毛细血管内皮损伤、通透性增高
    → 血浆纤维蛋白渗出至肺泡腔
    + TNF-α促进血浆蛋白渗出
    → 血浆蛋白凝聚为透明膜
    → 呼吸功能不全和肺损伤加重
    → 【恶性循环，病情越来越严重】
```

### 病理变化
- **肉眼**：双肺质地较坚实，色暗红，含气量少
- **镜下**：
  - 呼吸性细支气管、肺泡管和肺泡壁内表面贴附一层均质红染的**透明膜**
  - 所有肺叶均有不同程度的**肺不张**和**肺水肿**
  - 严重病例肺间质及肺泡腔内明显出血
  - 部分病例可见吸入的羊水成分（鳞状上皮细胞和角化物质）

## ❗ 易混点（ARDS vs NRDS）

| 鉴别点 | ARDS | NRDS |
|:-------|:-----|:------|
| **好发人群** | 成人（严重创伤/感染后） | **早产儿** |
| **核心机制** | 毛细血管+肺泡上皮损伤→通透性↑ | **肺表面活性物质缺乏** |
| **透明膜** | + | +（特征性） |
| **肺不张** | 不明显 | **明显** |
| **病死率** | 50%～60% | 高（预后差） |
