# 呼吸系统疾病

## 📌 概述
呼吸系统由**呼吸道**和**肺**构成。呼吸道以喉环状软骨为界分为上、下两部分。

### 解剖结构

```
上呼吸道：鼻、咽、喉
    ↓
下呼吸道：气管 → 支气管 → 小支气管 → 细支气管 → 终末细支气管（导气部）
    ↓
呼吸部：呼吸性细支气管 → 肺泡管 → 肺泡囊 → 肺泡
```

- **肺小叶**：3～5个终末细支气管连同分支及肺泡构成
- **肺腺泡**：Ⅰ级呼吸性细支气管及其远端肺组织，是肺的基本功能单位

### 防御机制
- **黏液-纤毛排送系统**（清除2～10µm颗粒）
- **肺泡巨噬细胞**（清除<2µm颗粒，分泌IFN-γ、TNF-α等）

## 🧠 核心临床推理链

### 推理链1：COPD → 肺心病（最核心）
```
慢性支气管炎 → 阻塞性通气障碍
    ↓
肺气肿（肺泡间隔破坏）
    ↓
肺毛细血管床减少 + 缺氧性肺小动脉痉挛
    ↓
肺血管构型重建（无肌细动脉肌化、中膜增厚）
    ↓
肺动脉高压（关键环节）
    ↓
右室负荷加重 → 右室肥大、扩张
    ↓
慢性肺源性心脏病 → 右心衰竭
```

### 推理链2：硅肺 → 肺心病
```
SiO₂粉尘吸入 → 硅结节 + 弥漫性肺纤维化
    ↓
肺毛细血管床减少 + 肺小动脉闭塞性脉管炎
    ↓
肺动脉高压 → 慢性肺源性心脏病
```

### 推理链3：急性肺炎的鉴别
```
起病急 + 发热 + 咳嗽
    ↓
X线表现
    ├── 大片致密阴影 → 大叶性肺炎（铁锈色痰）
    ├── 散在小片状阴影 → 小叶性肺炎（脓痰、湿啰音）
    ├── 间质改变为主 → 病毒性肺炎/支原体肺炎（干咳、少痰）
    └── 无特异性 → 军团菌肺炎（病情重、全身症状）
```
