# 急性气管支气管炎和细支气管炎

## 📌 定义
气管、支气管及细支气管黏膜的急性炎症，常见于儿童及老年人。

## 🔬 急性气管支气管炎（Acute Tracheobronchitis）

- **好发人群**：儿童及老年人
- **病因**：病毒感染基础上继发细菌感染（肺炎链球菌、流感嗜血杆菌、金葡菌等）

| 类型 | 病理变化 |
|:-----|:---------|
| **急性卡他性** | 黏膜及黏膜下层充血、水肿，少量中性粒细胞浸润，管腔覆稀薄黏性分泌物 |
| **急性化脓性** | 分泌物转为脓性，大量中性粒细胞浸润，可累及邻近肺泡 |
| **急性溃疡性** | 黏膜浅表性坏死、糜烂、溃疡形成（病毒感染合并化脓性炎） |

## 🔬 急性细支气管炎（Acute Bronchiolitis）

- **定义**：管径<2mm的细支气管急性炎症
- **好发**：4岁以下婴幼儿，**1岁以内约占90%**
- **病因**：呼吸道合胞病毒、腺病毒、副流感病毒

### 发病机制
```
婴幼儿小气道狭窄
    + 气流速度慢
    + 免疫功能不完善（IgA水平低）
    + 管壁无软骨支撑
    → 易阻塞
```

### 病理变化
- 黏膜充血肿胀，上皮坏死脱落
- 杯状细胞增多，黏液分泌增加
- 管腔内充满纤维蛋白、炎症细胞、脱落上皮细胞
- **后果**：管腔阻塞 → 小灶性肺萎缩 / 急性阻塞性肺气肿
- 严重者可形成**纤维闭塞性细支气管炎（bronchiolitis fibrosa obliterans）**

## ❗ 易混点
- 🚨 细支气管炎与支气管炎的区别：**管径<2mm**，无软骨支撑，易阻塞
- 🚨 急性细支气管炎**90%发生在1岁以内婴幼儿**
- 🚨 纤维闭塞性细支气管炎 = 严重细支气管炎的结局（机化）
