# 慢性支气管炎（Chronic Bronchitis）

## 📌 定义
发生于支气管黏膜及其周围组织的**慢性非特异性炎性疾病**。

> **诊断标准**：反复发作的咳嗽、咳痰或伴有喘息症状，每年至少持续**3个月**，连续**2年以上**
- **发病率**：中老年人群15%～20%
- **预后**：常并发[[肺气肿]]及[[慢性肺源性心脏病]]

## 👥 易感人群
- **中老年人**（>40岁），发病率15%～20%
- **长期吸烟者**（吸烟者患病率比不吸烟者高**2～10倍**，与吸烟量成正比）
- **冬春季多发**，北方寒冷地区高于南方
- **机体因素**：抵抗力降低、呼吸道防御功能受损者

## 🔬 病因

| 因素 | 具体说明 |
|:-----|:---------|
| **病毒和细菌感染** | 与感冒密切相关，冬春季多发；主要病毒：鼻病毒、腺病毒、呼吸道合胞病毒；主要细菌：肺炎链球菌、肺炎克雷伯菌、流感嗜血杆菌 |
| **吸烟** | 吸烟者患病率较不吸烟者高**2～10倍**，与吸烟量成正比；焦油、尼古丁、镉等损伤呼吸道黏膜 |
| **空气污染与过敏** | 工业烟雾、粉尘；喘息型患者常有过敏史 |
| **机体内在因素** | 抵抗力降低，防御功能受损，内分泌功能失调 |

## ⚙️ 机制

```
长期刺激因素（吸烟、感染、空气污染）
    ↓
呼吸道黏液-纤毛排送系统受损
    ↓
纤毛柱状上皮变性、坏死脱落 → 杯状细胞↑ → 鳞状上皮化生
    ↓
黏膜下腺体增生肥大 + 浆液性上皮黏液腺化生 → 分泌黏液增多
    ↓
管壁充血水肿，淋巴细胞、浆细胞浸润
    ↓
管壁平滑肌断裂、萎缩（喘息型：平滑肌束增生、肥大）
    ↓
软骨变性、萎缩或骨化
    ↓
反复发作 → 管壁纤维性增厚，管腔狭窄/纤维性闭锁
    ↓
炎症向管壁周围及肺泡扩展 → 细支气管炎和细支气管周围炎
    ↓
【引起慢性阻塞性肺气肿的病变基础】
```

## 🔬 病理变化（四大特征）

1. **黏液-纤毛排送系统受损**：纤毛柱状上皮变性、坏死脱落，杯状细胞增多，鳞状上皮化生
2. **腺体增生肥大**：黏膜下腺体增生肥大，浆液性上皮发生黏液腺化生 → 分泌黏液增多
3. **炎症细胞浸润**：管壁充血水肿，淋巴细胞、浆细胞浸润
4. **平滑肌和软骨改变**：平滑肌断裂、萎缩（喘息型增生肥大），软骨变性、萎缩或骨化

## 🩺 临床表现

| 症状 | 机制 |
|:-----|:-----|
| **咳嗽、咳痰** | 支气管黏膜受炎症刺激，黏液分泌增多；痰液白色黏液泡沫状；急性发作期黏液脓性或脓性痰 |
| **喘息** | 支气管痉挛或狭窄，黏液和渗出物阻塞管腔 |
| **哮鸣音、干/湿性啰音** | 听诊 |
| **痰量减少或无痰** | 慢性萎缩性气管炎（黏膜和腺体萎缩） |
| **阻塞性通气障碍** | 小气道狭窄和阻塞 → 呼气阻力增加 > 吸气 → 肺过度充气 → **并发肺气肿** |

## 🧠 临床推理链

```
慢性支气管炎
    ↓
阻塞性通气障碍 + 肺泡间隔破坏
    ↓
肺气肿
    ↓
肺循环阻力增加 → 肺动脉高压
    ↓
慢性肺源性心脏病
```
